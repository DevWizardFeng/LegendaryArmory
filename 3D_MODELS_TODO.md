# 3D模型替换清单

> **问题说明**: 当前的 `.glb` 模型文件都是简单的2D平面（只有4个顶点），上面贴了武器图片纹理，不是真正的3D模型。需要用Blender制作真正的3D武器模型来替换。

## 模型要求

- **格式**: glTF 2.0 (.glb)
- **位置**: 模型中心放置在原点 (0, 0, 0)
- **朝向**: 武器竖直放置，刀尖/枪尖朝上 (+Y方向)
- **大小**: 建议模型高度在 2-4 单位之间
- **材质**: 支持 PBR 材质 (baseColor, metallic, roughness)

## 文件路径

- **模型目录**: `entry/src/main/resources/rawfile/models/`
- **图片目录**: `entry/src/main/resources/rawfile/images/`

## Blender 导出设置建议

1. **格式**: glTF 2.0 Binary (.glb)
2. **包含**:
   - [x] Mesh
   - [x] Materials
   - [x] Textures (嵌入)
3. **变换**:
   - [x] +Y Up
   - [x] Apply Modifiers
4. **几何体**:
   - [x] Apply Modifiers
   - [x] UVs
   - [x] Normals
5. **材质**:
   - [x] Export Materials
   - 格式: PBR Metallic Roughness

## 注意事项

1. 模型中心应在原点 (0, 0, 0)
2. 武器竖直放置，尖端朝上 (+Y)
3. 模型大小适中（高度约 2-4 单位）
4. 确保有足够的多边形细节
5. 材质使用 PBR 工作流
6. 导出前检查法线方向
