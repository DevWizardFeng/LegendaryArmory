<h1 _ngcontent-nvy-c119="" class="doc-title ng-star-inserted" title="@ohos.app.ability.AbilityStage (AbilityStage组件管理器)"> @ohos.app.ability.AbilityStage (AbilityStage组件管理器) </h1>

<div _ngcontent-nvy-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>AbilityStage是一个<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/application-package-overview#应用的多module设计机制" target="_blank">Module</a>级别的组件管理器，用于进行Module级别的资源预加载、线程创建等初始化操作，以及维护Module下的应用状态。AbilityStage与Module一一对应，即一个Module拥有一个AbilityStage。</p>    <p>应用的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/hap-package" target="_blank">HAP</a>/<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/in-app-hsp" target="_blank">HSP</a>在首次加载时会创建一个AbilityStage实例。当一个Module中存在AbilityStage和其他组件（UIAbility/ExtensionAbility组件），AbilityStage实例会早于其他组件实例创建。</p>    <p>AbilityStage拥有<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-abilitystage#oncreate">onCreate()</a>、<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-abilitystage#ondestroy12">onDestroy()</a>生命周期回调和<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-abilitystage#onacceptwant">onAcceptWant()</a>、<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-abilitystage#onconfigurationupdate">onConfigurationUpdate()</a>、<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-abilitystage#onmemorylevel">onMemoryLevel()</a>事件回调等。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <p>本模块首批接口从API version 9 开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。</p>      <p>本模块接口仅可在Stage模型下使用。</p>     </div></div></div>    <div class="tiledSection">     <h2 id="导入模块"
