<h1 _ngcontent-pld-c119="" class="doc-title ng-star-inserted" title="@ohos.app.ability.InsightIntentDecorator (意图装饰器定义)"> @ohos.app.ability.InsightIntentDecorator (意图装饰器定义) </h1>

<div _ngcontent-pld-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>InsightIntentDecorator模块提供了几类意图装饰器，用于装饰类或方法。开发者可以<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/insight-intent-decorator-development" target="_blank">使用装饰器开发意图</a>, 将应用的功能定义为意图，并集成到智能问答、智能搜索、智能推荐等AI入口。</p>    <ul>     <li><a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#insightintentlink">@InsightIntentLink</a>装饰器：使用该装饰器装饰当前应用的uri链接，可以将该uri链接定义为意图，便于AI入口通过意图快速跳转到当前应用。该装饰器支持的参数参见<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#linkintentdecoratorinfo">LinkIntentDecoratorInfo</a>。</li>     <li><a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#insightintentpage">@InsightIntentPage</a>装饰器：使用该装饰器装饰当前应用的Page页面，可以将该Page页面定义为意图，便于AI入口通过意图快速跳转到当前Page页面。该装饰器支持的参数参见<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#pageintentdecoratorinfo">PageIntentDecoratorInfo</a>。</li>     <li><a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#insightintentfunction">@InsightIntentFunction</a>装饰器与<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#insightintentfunctionmethod">@InsightIntentFunctionMethod</a>装饰器：两者必须组合使用。使用<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#insightintentfunction">@InsightIntentFunction</a>装饰器来装饰类，同时使用<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#insightintentfunctionmethod">@InsightIntentFunctionMethod</a>装饰器来装饰类中的静态函数，可以将对应的静态函数定义为意图，便于AI入口能够快速执行此函数。</li>     <li><a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#insightintententry">@InsightIntentEntry</a>装饰器：使用该装饰器装饰一个继承自<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintententryexecutor">InsightIntentEntryExecutor</a>的类，实现意图操作并配置意图依赖的Ability组件，便于AI入口拉起依赖的Ability组件时，执行对应的意图操作。该装饰器支持的参数参见<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#entryintentdecoratorinfo">EntryIntentDecoratorInfo</a>。</li>     <li><a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#insightintentform">@InsightIntentForm</a>装饰器：使用该装饰器装饰<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-app-form-formextensionability">FormExtensionAbility</a>并配置FormExtensionAbility绑定的卡片名称，便于AI入口通过意图添加卡片。该装饰器支持的参数参见<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#formintentdecoratorinfo">FormIntentDecoratorInfo</a>。</li>     <li><a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#insightintententity">@InsightIntentEntity</a>装饰器：使用该装饰器装饰一个继承自<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintent#intententity20">IntentEntity</a>的类，可将该类定义为意图实体，用于传递意图调用时所需的参数。该装饰器支持的参数参见<a href="/consumer/cn/doc/harmonyos-references/js-apis-app-ability-insightintentdecorator#intententitydecoratorinfo">IntentEntityDecoratorInfo</a>。</li>    </ul>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <p>本模块首批接口从API version 20开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。</p>      <p>本模块接口仅可在Stage模型下使用。</p>     </div></div></div>    <div
