<h1 _ngcontent-vur-c119="" class="doc-title ng-star-inserted" title="@ohos.app.ability.UIExtensionAbility (带界面的ExtensionAbility组件)"> @ohos.app.ability.UIExtensionAbility (带界面的ExtensionAbility组件) </h1>

<div _ngcontent-vur-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>UIExtensionAbility组件是带界面的ExtensionAbility组件，继承自<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-app-ability-extensionability">ExtensionAbility</a>，提供了组件创建、销毁、前后台切换等基础生命周期。和UIAbility组件不同，UIExtensionAbility组件不会作为单独的任务在任务视图中体现。UIExtensionAbility组件被宿主窗口启动，该组件的前后台切换状态、以及是否可见均跟随宿主窗口。</p> <p>开发者不可以直接继承UIExtensionAbility组件，但可以根据实际业务场景选择使用继承自UIExtensionAbility组件的其他组件。例如，开发者处理其他应用分享的数据时，可以使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-app-ability-shareextensionability">ShareExtensionAbility组件</a>；开发者提供卡片编辑功能时，可以使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-app-form-formeditextensionability">FormEditExtensionAbility组件</a>。</p> <p>各类Ability组件的继承关系详见<a
