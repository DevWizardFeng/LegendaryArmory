<h1 _ngcontent-aoq-c119="" class="doc-title ng-star-inserted" title="@ohos.util.Deque (线性容器Deque)"> @ohos.util.Deque (线性容器Deque) </h1>

<div _ngcontent-aoq-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>Deque（double ended queue）基于循环队列的数据结构实现，支持两端元素的插入和删除，同时具备先进先出以及先进后出的特点。Deque会根据实际需要动态调整容量，每次扩容两倍。</p>    <p>Deque和<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-queue">Queue</a>相比，Deque允许在两端执行插入和删除操作，Queue只能在头部删除元素，尾部插入元素。</p>    <p>与<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arraylist">ArrayList</a>相比，它们都支持在两端插入和删除元素，但Deque不支持中间插入。Deque在头部插入删除元素的效率高于ArrayList，而ArrayList随机访问元素的效率高于Deque。</p>    <p><strong>推荐使用场景：</strong> 需要在集合两端频繁增删元素时，推荐使用Deque。</p>    <p>文档中使用了泛型，涉及以下泛型标记符：</p>    <ul>     <li>T：Type，类</li>    </ul>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div
