<h1 _ngcontent-tga-c119="" class="doc-title ng-star-inserted" title="@ohos.util.HashMap (非线性容器HashMap)"> @ohos.util.HashMap (非线性容器HashMap) </h1>

<div _ngcontent-tga-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>HashMap底层采用数组、链表和红黑树实现，支持高效查询、插入和删除。HashMap存储内容基于key-value的键值对映射，不允许重复的key，且一个key只能对应一个value。</p>    <p>HashMap和<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-treemap">TreeMap</a>相比，HashMap依据键的hashCode存取数据，访问速度较快。而TreeMap是有序存取，效率较低。</p>    <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-hashset">HashSet</a>基于HashMap实现。HashMap的输入参数由key、value两个值组成。在HashSet中，只对value对象进行处理。</p>    <p><strong>推荐使用场景：</strong> 需要快速存取、删除以及插入键值对数据时，推荐使用HashMap。</p>    <p>文档中使用了泛型，包含以下泛型标记符：</p>    <ul>     <li>      <p>K：Key，键</p></li>     <li>V：Value，值</li>    </ul>    <div><div class="hw-editor-tip info"><div
