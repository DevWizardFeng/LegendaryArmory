<h1 _ngcontent-mru-c119="" class="doc-title ng-star-inserted" title="@ohos.util.LightWeightMap (非线性容器LightWeightMap)"> @ohos.util.LightWeightMap (非线性容器LightWeightMap) </h1>

<div _ngcontent-mru-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>LightWeightMap可用于存储具有关联关系的key-value键值对集合，存储元素中key值唯一，每个key对应一个value。</p>    <p>LightWeightMap依据泛型定义，采用轻量级结构，初始默认容量大小为8，每次扩容大小为原始容量的两倍。</p>    <p>集合中key值的查找依赖于hash算法，通过一个数组存储hash值，然后映射到其他数组中的key值及value值。</p>    <p>LightWeightMap和<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-hashmap">HashMap</a>都是用来存储键值对的集合，但LightWeightMap占用内存更小。</p>    <p><strong>推荐使用场景：</strong> 当需要存取key-value键值对时，推荐使用占用内存更小的LightWeightMap。</p>    <p>文档中使用了泛型，涉及以下泛型标记符：</p>    <ul>     <li>      <p>K：Key，键</p></li>     <li>V：Value，值</li>    </ul>    <div><div class="hw-editor-tip info"><div
