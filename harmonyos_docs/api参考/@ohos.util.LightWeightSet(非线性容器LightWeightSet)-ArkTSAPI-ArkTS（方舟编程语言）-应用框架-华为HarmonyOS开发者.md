<h1 _ngcontent-odf-c119="" class="doc-title ng-star-inserted" title="@ohos.util.LightWeightSet (非线性容器LightWeightSet)"> @ohos.util.LightWeightSet (非线性容器LightWeightSet) </h1>

<div _ngcontent-odf-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>LightWeightSet可用于存储一系列值的集合，存储元素中value值唯一。</p>    <p>LightWeightSet依据泛型定义，采用轻量级结构，初始默认容量大小为8，每次扩容大小为原始容量的两倍。</p>    <p>集合中value值的查找依赖于hash算法，通过一个数组存储hash值，然后映射到其他数组中的value值。</p>    <p>LightWeightSet和<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-hashset">HashSet</a>都是用来存储键值的集合，但LightWeightSet的占用内存更小。</p>    <p><strong>推荐使用场景：</strong> 当需要存取某个集合或是对某个集合去重时，推荐使用占用内存更小的LightWeightSet。</p>    <p>文档中使用了泛型，涉及以下泛型标记符：</p>    <ul>     <li>T：Type，类</li>    </ul>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div
