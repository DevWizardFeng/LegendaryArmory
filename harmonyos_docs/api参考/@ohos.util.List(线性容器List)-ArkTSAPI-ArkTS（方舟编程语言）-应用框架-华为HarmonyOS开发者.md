<h1 _ngcontent-pey-c119="" class="doc-title ng-star-inserted" title="@ohos.util.List (线性容器List)"> @ohos.util.List (线性容器List) </h1>

<div _ngcontent-pey-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>List底层通过单向链表实现，每个节点有一个指向后一个元素的引用。查询元素必须从头遍历，因此查询效率低，但插入和删除效率高。List允许元素为null。</p>    <p>List和<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-linkedlist">LinkedList</a>相比，LinkedList是双向链表，可以快速地在头尾进行增删，而List是单向链表，不支持双向操作。</p>    <div><div class="hw-editor-tip warn"><div class="title">注意</div><div class="content">      <p>在List中使用[index]的方式获取元素可能导致未定义结果，推荐使用get()方法。</p>     </div></div></div>    <p><strong>推荐使用场景：</strong> 当需要频繁的插入删除元素且需要使用单向链表时，推荐使用List。</p>    <p>文档使用了泛型，涉及以下泛型标记符：</p>    <ul>     <li>T：Type，类</li>    </ul>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">
