<h1 _ngcontent-dtd-c119="" class="doc-title ng-star-inserted" title="@ohos.util.PlainArray (非线性容器PlainArray)"> @ohos.util.PlainArray (非线性容器PlainArray) </h1>

<div _ngcontent-dtd-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>PlainArray可用于存储具有关联关系的key-value键值对集合，其中key值唯一且类型为number，每个key对应一个value。</p>    <p>PlainArray依据泛型定义，采用轻量级结构，通过二分查找算法在集合中查找 key 值，并映射到其他数组中的value值。</p>    <p>PlainArray和<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-lightweightmap">LightWeightMap</a>都是用来存储键值对，且均采用轻量级结构，但PlainArray的key值类型仅限于number。</p>    <p><strong>推荐使用场景：</strong> 当需要存储key值为number类型的键值对时，可以使用PlainArray。</p>    <p>文档中使用了泛型，涉及以下泛型标记符：</p>    <ul>     <li>T：Type，类</li>    </ul>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div
