<h1 _ngcontent-qoq-c119="" class="doc-title ng-star-inserted" title="@ohos.util.Stack (线性容器Stack)"> @ohos.util.Stack (线性容器Stack) </h1>

<div _ngcontent-qoq-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>Stack基于数组的数据结构实现，特点是先进后出，只能在一端进行数据的插入和删除。</p>    <p>Stack和<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-queue">Queue</a>相比，Queue基于循环队列实现，只能在一端删除另一端插入，而Stack都在一端操作。</p>    <p><strong>推荐使用场景：</strong> 一般符合先进后出的场景可以使用Stack。</p>
