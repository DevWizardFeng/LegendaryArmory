<h1 _ngcontent-esm-c119="" class="doc-title ng-star-inserted" title="@ohos.util.TreeMap (非线性容器TreeMap)"> @ohos.util.TreeMap (非线性容器TreeMap) </h1>

<div _ngcontent-esm-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>TreeMap可用于存储具有关联关系的key-value键值对集合，存储元素中key值唯一，每个key对应一个value。</p>    <p>TreeMap底层使用红黑树实现，可以利用二叉树特性快速查找键值对。key值有序存储，可以实现快速的插入和删除。</p>    <p>TreeMap和<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-hashmap">HashMap</a>相比，HashMap依据键的hashCode存取数据，访问速度较快。而TreeMap是有序存取，效率较低。</p>    <p><strong>推荐使用场景：</strong> 一般需要存储有序键值对的场景，可以使用TreeMap。</p>    <p>文档使用了泛型，涉及以下泛型标记符：</p>    <ul>     <li>      <p>K：Key，键</p></li>     <li>      <p>V：Value，值</p></li>    </ul>    <div><div class="hw-editor-tip info"><div
