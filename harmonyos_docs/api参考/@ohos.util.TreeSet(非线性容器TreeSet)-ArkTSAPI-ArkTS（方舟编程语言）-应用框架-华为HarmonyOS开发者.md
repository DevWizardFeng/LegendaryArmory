<h1 _ngcontent-hii-c119="" class="doc-title ng-star-inserted" title="@ohos.util.TreeSet (非线性容器TreeSet)"> @ohos.util.TreeSet (非线性容器TreeSet) </h1>

<div _ngcontent-hii-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>TreeSet基于<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-treemap">TreeMap</a>实现，在TreeSet中，只对value对象进行处理。TreeSet可用于存储一系列值的集合，元素中value唯一且有序。</p>    <p>TreeSet和<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-hashset">HashSet</a>中的元素都不允许重复。HashSet中的数据无序存放，而TreeSet是有序存放。HashSet允许插入null值，但TreeSet不建议插入空值，可能会影响排序结果。</p>    <p><strong>推荐使用场景：</strong> 一般需要存储有序集合的场景，可以使用TreeSet。</p>    <p>文档中使用了泛型，涉及以下泛型标记符：</p>    <ul>     <li>T：Type，类</li>    </ul>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div
