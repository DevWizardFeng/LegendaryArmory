<p><strong>可能原因</strong></p> <ol><li>设置导航状态时目的地或途经点的名字的长度超出1024字节。</li><li>设置导航元数据时当前道路名或下一次进入的道路名的长度超出1024字节。</li><li>参数值超出范围，比如设置导航状态时经纬度值不在有效范围（纬度值的数值范围是[-90, 90]，经度值的有效范围是[-180, 180]）内。</li></ol>
