<ol><li><p>由于与属性方法同时设置或者在AttributeUpdater中实现applyNormalAttribute等方法时，涉及到与状态管理更新机制同时使用，易出现混淆，因此不建议在同一组件上同时用两种方法设置相同属性。</p>  </li><li><p>当与属性方法同时设置时，属性生效的原则为：后设置的生效。</p> <p> 若先进行属性直通更新，后通过状态管理机制更新属性方法，则后更新的属性方法生效；</p> <p> 若先通过状态管理机制更新属性方法，后进行属性直通更新，则属性直通更新生效。</p>  </li><li><p>一个AttributeUpdater对象只能同时关联一个组件，否则将出现设置的属性只在一个组件上生效的现象。</p>  </li><li><p>开发者需要自行保障AttributeUpdater中T和C的类型匹配。比如T为ImageAttribute，C要对应为ImageInterface，否则可能导致</p> <p> 使用updateConstructorParams时功能异常。</p>  </li><li><p>updateConstructorParams当前只支持Button，Image，Text，Span，SymbolSpan和ImageSpan组件。</p>  </li><li><p>AttributeUpdater不支持深浅色切换等状态管理相关的操作。</p>  </li></ol>  </div></div></div> </div>  <div
