<h1 _ngcontent-lcc-c119="" class="doc-title ng-star-inserted" title="EmbeddedComponent"> EmbeddedComponent </h1>

<div _ngcontent-lcc-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>EmbeddedComponent用于支持在当前页面嵌入本应用内其他<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-app-ability-embeddeduiextensionability">EmbeddedUIExtensionAbility</a>提供的UI。EmbeddedUIExtensionAbility在独立进程中运行，完成页面布局和渲染。</p>    <p>通常用于有进程隔离诉求的模块化开发场景。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <p>该组件从API Version 12开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</p>
