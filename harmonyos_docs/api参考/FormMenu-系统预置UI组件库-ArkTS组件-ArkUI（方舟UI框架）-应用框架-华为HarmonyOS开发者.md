<h1 _ngcontent-neg-c119="" class="doc-title ng-star-inserted" title="FormMenu"> FormMenu </h1>

<div _ngcontent-neg-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>本组件封装了一个“添加至桌面”菜单，用于实现应用内长按组件生成“添加至桌面”菜单，点击该菜单，触发卡片添加至桌面操作。通过桌面访问该应用快捷卡片，可以直接访问该组件功能。在应用使用过程中，该组件作为留存和复访入口，可吸引用户将功能快捷添加到桌面。</p>    <p>本组件支持应用内长按菜单快捷添加卡片到桌面：</p>    <ol>     <li>      <p>开发者将卡片数据以及应用内功能组件ID传给卡片框架。</p></li>     <li>      <p>点击事件会根据组件ID获取应用内功能组件的快照和位置，用于添加到桌面时的过渡动效。</p></li>     <li>      <p>卡片框架通过将加桌数据通知给桌面，触发卡片添加到桌面操作。</p></li>    </ol>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <p>该组件从API Version 12开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</p>      <p>该组件不支持在Wearable设备上使用。</p>      <p>卡片具体开发指导请参考<a
