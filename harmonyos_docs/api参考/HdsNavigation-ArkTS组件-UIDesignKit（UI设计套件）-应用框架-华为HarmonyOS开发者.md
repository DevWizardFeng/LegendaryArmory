<div><p>本模块提供导航组件的能力，默认支持标题栏随内容区滚动的动态模糊样式。6.0.0(20)版本以后，推荐使用<a href="/consumer/cn/doc/harmonyos-references/ui-design-hdsnavigation#section14512122441412">bindToScrollable</a>、<a href="/consumer/cn/doc/harmonyos-references/ui-design-hdsnavigation#section1375318442179">bindToNestedScrollable</a>属性绑定导航组件和可滚动容器组件后，再使用导航组件滚动相关的功能，从而获得更优的体验。如滚动生效动态模糊样式，标题栏随内容区滚动动态显隐功能等。</p>
