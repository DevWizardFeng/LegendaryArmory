<div _ngcontent-ggh-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>当今的软件市场是一个全球性的市场，应用程序的开发和维护需要支持多种语言。International Components for Unicode (ICU) 库为多平台多系统提供了强大且功能齐全的多语言文化信息处理能力。</p> <p><a href="https://unicode-org.github.io/icu/userguide/icu4c/" target="_blank">ICU4C</a>是ICU 库的C语言版本，部分接口已开放，支持如下能力：</p> <ul><li>区域与文化标识(uloc.h)</li><li>时间日期格式化(udat.h)</li><li>数字及度量衡格式化(unumberformatter.h，unum.h)</li><li>日历及本地历法(ucal.h)</li><li>排序(ucol.h)</li><li>断词换行(ubrk.h)</li><li>文本正规化(unorm2.h)</li><li>音译(utrans.h)</li><li>字符/文本属性(uchar.h，uscript.h)</li><li>编码转换(ucnv.h)</li><li>文本双向处理(ubidi.h)</li><li>域名转换(uidna.h)</li><li>Unicode字符串处理(ustring.h)</li><li>ICU文本抽象表示(utext.h)</li></ul>
