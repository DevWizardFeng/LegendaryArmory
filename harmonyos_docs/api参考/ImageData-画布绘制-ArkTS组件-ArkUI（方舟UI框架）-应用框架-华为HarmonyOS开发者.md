<h1 _ngcontent-jks-c119="" class="doc-title ng-star-inserted" title="ImageData"> ImageData </h1>

<div _ngcontent-jks-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>ImageData对象可以存储canvas渲染的像素数据。</p> <div><div class="hw-editor-tip <p>创建ImageData时，宽高不超过16384px，最大面积不超过16000px*16000px，超过最大面积则无法正常绘制。当创建面积超过536870911px时，返回值的width和height均为0px，data为undefined。</p>  </div></div></div>  <div class="tiledSection"><h2 id="constructor"
