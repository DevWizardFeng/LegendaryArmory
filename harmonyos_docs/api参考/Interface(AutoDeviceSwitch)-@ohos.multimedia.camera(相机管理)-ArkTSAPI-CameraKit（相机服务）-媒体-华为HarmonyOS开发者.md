<h1 _ngcontent-jah-c119="" class="doc-title ng-star-inserted" title="Interface (AutoDeviceSwitch)"> Interface (AutoDeviceSwitch) </h1>

<div _ngcontent-jah-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>AutoDeviceSwitch 继承自 <a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-autodeviceswitchquery">AutoDeviceSwitchQuery</a>。</p> <p>自动切换镜头类，继承自<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-autodeviceswitchquery">AutoDeviceSwitchQuery</a>，用于使能或去使能自动切换镜头。</p> <p>使用建议：自动切换镜头功能由系统自动完成输入设备切换、会话配置和参数接续，</p> <p>如系统发现镜头切换时，两颗镜头的变焦范围不一致，则会通过<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-i#autodeviceswitchstatus13">AutoDeviceSwitchStatus</a>中的isDeviceCapabilityChanged字段告知应用，</p> <p>但仍需要应用自己处理UX的变更（如变焦范围的调整，需要重新通过<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-zoomquery#getzoomratiorange11">getZoomRatioRange</a>接口获取数据并更新UX），</p>
