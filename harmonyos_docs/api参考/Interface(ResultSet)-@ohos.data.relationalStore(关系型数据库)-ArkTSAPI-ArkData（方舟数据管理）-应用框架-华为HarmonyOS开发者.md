<h1 _ngcontent-crk-c119="" class="doc-title ng-star-inserted" title="Interface (ResultSet)"> Interface (ResultSet) </h1>

<div _ngcontent-crk-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>提供通过查询数据库生成的数据库结果集的访问方法。结果集是指用户调用关系型数据库查询接口之后返回的结果集合，提供了多种灵活的数据访问方式，以便用户获取各项数据。</p>    <p>ResultSet实例不会实时刷新。使用结果集后，如果数据库中的数据发生变化（如增删改操作），需要重新查询才能获取到最新的数据。</p>    <p>下列API示例中，都需先使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-data-relationalstore-rdbstore#query">query</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-data-relationalstore-rdbstore#querysql">querySql</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-data-relationalstore-rdbstore#remotequery-1">remoteQuery</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-data-relationalstore-rdbstore#querylockedrow12">queryLockedRow</a>等query类方法中任一方法获取到ResultSet实例，再通过此实例调用对应方法。</p>
