Kit使用，支持同时出普通预览流和安全流的业务场景。请参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/camera-secure-photo" target="_blank">安全相机开发指导</a>。</p>    <div><div class="hw-editor-tip info"><div
