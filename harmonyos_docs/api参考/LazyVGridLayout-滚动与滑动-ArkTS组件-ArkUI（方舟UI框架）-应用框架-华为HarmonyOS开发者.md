<h1 _ngcontent-sml-c119="" class="doc-title ng-star-inserted" title="LazyVGridLayout"> LazyVGridLayout </h1>

<div _ngcontent-sml-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>该组件用于实现支持懒加载的网格布局，其父组件仅限于<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-waterflow">WaterFlow</a>或<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-flowitem">FlowItem</a>，并支持使用自定义组件、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-nodecontainer">NodeContainer</a>组件封装后，在WaterFlow或FlowItem组件下应用。</p> <p>该组件仅在WaterFlow组件的单列模式或分段布局中的单列分段，并且布局方向为FlexDirection.Column的情况下支持懒加载。在WaterFlow的多列模式或布局方向为FlexDirection.Row或FlexDirection.RowReverse的情况下使用该组件，则不支持懒加载。此外，在布局方向为FlexDirection.ColumnReverse的WaterFlow组件下使用该组件会导致显示异常。当懒加载功能生效时，该组件仅加载WaterFlow显示区域内的子组件，并在帧间空闲时隙预加载显示区域上方和下方各半屏的内容。</p> <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content"> <ul><li>该组件从API version 19开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</li><li>LazyVGridLayout组件高度默认自适应内容，不建议设置高度、高度约束或宽高比，设置后会导致显示异常。</li></ul>  </div></div></div>  <div class="tiledSection"><h2 id="接口" <p>设置为'0fr'时，该列的列宽为0，不显示子组件。设置为其他非法值时，子组件显示为固定1列。</p> <p><strong>元服务API：</strong> 从API version 19开始，该接口支持在元服务中使用。</p> <p><strong>系统能力：</strong> SystemCapability.ArkUI.ArkUI.Full</p>
