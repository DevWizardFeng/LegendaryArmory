class="support-device-item">Wearable</span></div><p>startAbilityByLiveForm(want: Want): Promise&lt;void&gt;</p>     <p>拉起互动卡片提供方（应用）的页面，使用Promise异步回调。</p>     <p>该接口仅支持拉起互动卡片提供方（应用）的页面，不支持拉起其他应用的页面，否则会抛出错误码16501011。</p>     <p>该接口建议在点击事件回调中调用，不建议在其他手势事件回调中调用，不支持非手势事件直接调用，否则会抛出错误码16501011。</p>     <p>另外，在点击事件回调中，支持直接调用，不支持延时后调用，否则会抛出错误码16501011。</p>     <p><strong>模型约束：</strong> 此接口仅可在Stage模型下使用。</p>     <p><strong>系统能力：</strong> SystemCapability.Ability.Form</p>     <p><strong>元服务API：</strong> 从API version 20开始，该接口支持在元服务中使用。</p>
