<div _ngcontent-fdh-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>跑马灯组件，用于滚动展示一段单行文本。仅当文本内容宽度大于等于跑马灯组件宽度时滚动，当文本内容宽度小于跑马灯组件宽度时不滚动。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <p>该组件从API version 8开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</p>      <p>为了不影响滚动帧率，建议在滚动类组件中Marquee的个数不超过4个，或者使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-text">Text</a>组件的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-appendix-enums#textoverflow">TextOverflow.MARQUEE</a>替代。</p>      <p>对于Marquee组件动态帧率的场景，可以使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-marqueedynamicsyncscene">MarqueeDynamicSyncScene</a>接口实现。</p>      <p>在文本宽度小于跑马灯组件宽度时，使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-animatorproperty">属性动画</a>实现滚动。</p>     </div></div></div>    <div class="tiledSection">     <h2 id="子组件"
