<h1 _ngcontent-gug-c119="" class="doc-title ng-star-inserted" title="NodeContainer"> NodeContainer </h1>

<div _ngcontent-gug-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>基础组件，用于挂载自定义节点（如<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-framenode">FrameNode</a>或<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-buildernode">BuilderNode</a>），并通过<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-nodecontroller">NodeController</a>动态控制节点的上树和下树。组件不支持尾随添加子节点，接受一个<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-nodecontroller">NodeController</a>实例接口，需与NodeController组合使用。</p> <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content"> <p>该组件从API version 11开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</p>  <p>该组件下仅支持挂载自定义节点<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-framenode">FrameNode</a>或者是<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-buildernode">BuilderNode</a>中获取的根节点FrameNode。</p> <p>不支持挂载查询获得的系统组件<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-framenode#ismodifiable12">代理节点</a>。</p>  <p>当前不支持使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-attribute-modifier">动态属性设置</a>。</p>  <p>该组件下的节点树构建中会使用UI实例<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext">UIContext</a>，当实例切换时可能会因为实例不匹配而出现问题，因此该组件当前不支持跨实例的节点复用。</p>  <p>该组件未销毁时，不会主动触发挂载节点的下树。</p>
