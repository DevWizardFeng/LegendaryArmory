<div _ngcontent-jga-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>为组件绑定Popup气泡，并设置气泡内容、交互逻辑和显示状态。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <ul>       <li>        <p>从API version 7开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</p></li>       <li>        <p>Popup气泡的显示状态在onStateChange事件回调中反馈，其显隐与组件的创建或销毁无强对应关系。</p></li>       <li>        <p>Popup气泡的高度为当前窗口高度 - 上下安全区域高度（状态栏、导航条）- 80vp。</p></li>       <li>        <p>多个气泡同时弹出时，子窗内显示的气泡比主窗内显示的气泡层级高，所处窗口相同时，后面弹出的气泡层级比先弹出的气泡层级高。</p></li>       <li>        <p>2in1设备默认有双描边，其他设备默认无双描边。</p></li>       <li>        <p>子窗弹窗里不能再弹出子窗弹窗，例如bindPopup设置了showInSubWindow为true时，则不能再弹出另一个设置了showInSubWindow为true的弹窗。</p></li>      </ul>     </div></div></div>    <div class="tiledSection">     <h2 id="bindpopup"
