<h1 _ngcontent-xtd-c119="" class="doc-title ng-star-inserted" title="PushExtensionAbility（推送扩展Ability）"> PushExtensionAbility（推送扩展Ability） </h1>

<div _ngcontent-xtd-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <p>推送扩展Ability目前为预留能力，暂未开放使用。若您的应用有拉起应用的子进程，在子进程中自行处理业务的诉求，请参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/push-extend-noti" target="_blank">推送通知扩展消息</a>。</p>     </div></div></div>    <p>PushExtensionAbility为推送扩展Ability，提供获取场景化消息数据和生命周期销毁的回调。有如下约束：</p>    <ul>     <li><span style="color: rgb(29,29,26);">PushExtensionAbility为独立子进程，轻量级，不允许唤醒主进程</span>。</li>     <li><span style="color: rgb(29,29,26);">不允许调用通知API、卡片API</span>、窗口API、弹窗API、实况窗API。</li>     <li>生命周期根据场景受控，默认小于10秒，超过10秒子进程生命周期结束。</li>    </ul>    <p><strong>模型约束：</strong>此接口仅可在Stage模型下使用。</p>    <p><strong>系统能力：</strong>SystemCapability.Push.PushService</p>
