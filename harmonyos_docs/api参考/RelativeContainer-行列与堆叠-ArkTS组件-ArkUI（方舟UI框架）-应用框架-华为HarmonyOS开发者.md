<h1 _ngcontent-tfn-c119="" class="doc-title ng-star-inserted" title="RelativeContainer"> RelativeContainer </h1>

<div _ngcontent-tfn-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>相对布局组件，用于复杂场景中元素对齐的布局。</p>    <p>子组件可以通过设置<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-location#alignrules9">alignRules</a>来设置自身在相对容器中的对齐规则。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <ul>       <li>该组件从API version 9开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</li>       <li>在RelativeContainer组件中，不设置<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-size#width">width</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-size#height">height</a>时，对应属性布局表现与设置为100%相同。</li>       <li>从API version 11开始，在RelativeContainer组件中，<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-size#width">width</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-size#height">height</a>设置"auto"表示自适应子组件。当width设置"auto"时，如果水平方向上子组件以容器作为锚点，则"auto"不生效（即视为不设置width），垂直方向上同理。</li>       <li>从API version 20开始，在RelativeContainer组件中，<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-size#width15">width</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-size#height15">height</a>设置LayoutPolicy.wrapContent表示自适应子组件且被祖先节点尺寸约束，设置LayoutPolicy.fixAtIdealSize表示自适应子组件且不被祖先节点尺寸约束。当width设置wrapContent或fixAtIdealSize时，如果水平方向上子组件直接或间接以容器作为锚点，则容器在该方向上的尺寸不自适应该组件，垂直方向上同理。</li>       <li>相对布局容器内的子组件的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-size#margin">margin</a>含义不同于通用属性的margin，其含义为到该方向上的锚点的距离。若该方向上没有锚点，则该方向的margin不生效。</li>      </ul>
