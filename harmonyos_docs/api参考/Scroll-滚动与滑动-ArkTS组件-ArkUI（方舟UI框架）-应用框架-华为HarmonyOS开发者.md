<h1 _ngcontent-srf-c119="" class="doc-title ng-star-inserted" title="Scroll"> Scroll </h1>

<div _ngcontent-srf-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted renderOptimization" style="position: relative;">   <div>    <p>可滚动的容器组件，当子组件的布局尺寸超过父组件的尺寸时，内容可以滚动。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <ul>       <li>该组件从API version 7开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</li>       <li>该组件嵌套List子组件滚动时，若List不设置宽高，则默认全部加载，在对性能有要求的场景下建议指定List的宽高，最佳实践请参考<a href="https://developer.huawei.com/consumer/cn/doc/best-practices/bpta-lazyforeach-optimization#section6296154115367" target="_blank">懒加载优化性能-Scroll嵌套List导致按需加载失效</a>。</li>       <li>该组件滚动的前提是主轴方向大小小于内容大小。</li>       <li>Scroll组件<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-sharp-clipping#clip12">通用属性clip</a>的默认值为true。</li>       <li>Scroll组件的高度超出屏幕显示范围时，可以通过设置通用属性<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-size#layoutweight">layoutWeight</a>让Scroll高度适应主轴的剩余空间。</li>       <li>手指触摸屏幕时，会停止当前触摸范围内所有滚动组件的滚动动画（<a href="/consumer/cn/doc/harmonyos-references/ts-container-scroll#scrollto">scrollTo</a>和<a href="/consumer/cn/doc/harmonyos-references/ts-container-scroll#scrolltoindex">scrollToIndex</a>接口触发的滚动动画除外），包括边缘回弹动画。</li>       <li>组件内部已绑定手势实现跟手滚动等功能，需要增加自定义手势操作时请参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-gesture-blocking-enhancement">手势拦截增强</a>进行处理。</li>      </ul>     </div></div></div>    <div class="tiledSection
