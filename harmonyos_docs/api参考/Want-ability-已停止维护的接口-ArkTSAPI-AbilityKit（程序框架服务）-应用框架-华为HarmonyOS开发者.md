<h1 _ngcontent-anl-c119="" class="doc-title ng-star-inserted" title="Want"> Want </h1>

<div _ngcontent-anl-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>Want是对象间信息传递的载体, 可以用于应用组件间的信息传递。 Want的使用场景之一是作为<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-application-uiabilitycontext#startability">startAbility</a>的参数, 其包含了指定的启动目标, 以及启动时需携带的相关数据, 如bundleName和abilityName字段分别指明目标Ability所在应用的Bundle名称以及对应包内的Ability名称。当Ability A需要启动Ability B并传入一些数据时, 可使用Want作为载体将这些数据传递给Ability B。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <p>本模块首批接口从API version 6开始支持，从API version 9废弃，使用<a
