<h1 _ngcontent-dbt-c119="" class="doc-title ng-star-inserted" title="loadNativeModule (同步动态加载系统库接口)"> loadNativeModule (同步动态加载系统库接口) </h1>

<div _ngcontent-dbt-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>本模块提供了同步动态加载系统库接口的能力。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <p>本模块首批接口从API version 12开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。</p>      <p>本模块接口仅可在Stage模型下使用。</p>     </div></div></div>    <div class="tiledSection">     <h2 id="loadnativemodule" device-type="phone,2in1,tablet,wearable,tv">loadNativeModule<i class="anchor-icon anchor-icon-device active" active-tips="隐藏支持设备" class="support-device-item">Wearable</span></div><p>loadNativeModule(moduleName: string): Object</p>     <p>loadNativeModule接口用于同步动态加载native模块，目的是按需加载所需要的模块。使用该接口会增加加载so文件的时间，开发者需评估其对功能的影响。</p>     <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">       <p>loadNativeModule加载的模块名指的是依赖方oh-package.json5文件的dependencies字段内的名称。</p>       <p>loadNativeModule接口仅支持在UI主线程中加载native模块。</p>       <p>该接口在使用常量字符串或变量表达式作为参数时，都需要配置依赖。</p>      </div></div></div>     <p><strong>系统能力</strong>：SystemCapability.Utils.Lang</p>
