<p>用于卡片内部和提供方应用间的交互，当前支持router、message和call三种类型的事件，仅在卡片中可以调用。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <p>本接口从API version 9开始支持。</p>
