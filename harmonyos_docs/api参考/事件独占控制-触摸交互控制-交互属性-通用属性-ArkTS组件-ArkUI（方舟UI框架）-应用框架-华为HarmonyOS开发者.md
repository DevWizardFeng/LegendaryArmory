<h1 _ngcontent-cqp-c119="" class="doc-title ng-star-inserted" title="事件独占控制"> 事件独占控制 </h1>

<div _ngcontent-cqp-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>设置组件是否独占事件，事件范围包括组件自带的事件和开发者自定义的点击、触摸、手势事件。</p> <p>在一个窗口内，设置了独占控制的组件上的事件如果首先响应，则本次交互只允许此组件上设置的事件响应，窗口内其他组件上的事件不会响应。</p> <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content"> <p> 从API version 11开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</p>  </div></div></div>  <div class="tiledSection"><h2 id="monopolizeevents" device-type="phone,2in1,tablet,wearable,tv">monopolizeEvents<i class="anchor-icon
