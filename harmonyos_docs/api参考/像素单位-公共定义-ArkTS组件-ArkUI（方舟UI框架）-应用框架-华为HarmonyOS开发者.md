<h1 _ngcontent-xgd-c119="" class="doc-title ng-star-inserted" title="像素单位"> 像素单位 </h1>

<div _ngcontent-xgd-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>ArkUI为开发者提供4种像素单位，采用vp为基准数据单位。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <ul>       <li>        <p>本模块首批接口从API version 7开始支持，后续版本的新增接口，采用上角标单独标记接口的起始版本。</p></li>       <li>        <p>直接使用vp2px/px2vp/fp2px/px2fp/lpx2px/px2lpx可能存在<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-global-interface#ui上下文不明确" target="_blank">UI上下文不明确</a>的问题，以下接口从API version 18开始废弃，建议使用getUIContext()获取<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext">UIContext</a>实例，再使用UIContext下的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#vp2px12">vp2px</a>/<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#px2vp12">px2vp</a>/<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#fp2px12">fp2px</a>/<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#px2fp12">px2fp</a>/<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#lpx2px12">lpx2px</a>/<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-uicontext-uicontext#px2lpx12">px2lpx</a>调用绑定实例的接口。</p></li>       <li>        <p>在UI实例未创建时，vp2px/px2vp使用默认屏幕的虚拟像素比进行转换。在该场景下，使用UIContext接口替换时，开发者可参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-global-interface#像素单位转换接口替换为uicontext接口" target="_blank">像素单位转换接口替换为UIContext接口</a>。</p></li>      </ul>     </div></div></div>    <div class="tablenoborder">     <div class="tbBox"><table class="tableClass2_t">                   <tbody><tr class="firstRow">        <th align="left" class="cellrowborder" id="mcps1.3.3.1.3.1.1" width="50%">名称</th>        <th align="left" class="cellrowborder" id="mcps1.3.3.1.3.1.2" width="50%">描述</th>       </tr><tr>        <td class="cellrowborder" width="50%">px</td>        <td class="cellrowborder" width="50%">屏幕物理像素单位。</td>       </tr>       <tr>        <td class="cellrowborder" width="50%">vp</td>        <td class="cellrowborder" width="50%">         <p>屏幕密度相关像素，根据屏幕像素密度转换为屏幕物理像素，当数值不带单位时，默认单位vp。</p>         <p><strong>说明：</strong></p>         <p>vp与px的比例与屏幕像素密度有关。</p></td>       </tr>       <tr>        <td class="cellrowborder" width="50%">fp</td>        <td class="cellrowborder" width="50%">字体像素，与vp类似适用屏幕密度变化，随系统字体大小设置变化。</td>       </tr>       <tr>        <td class="cellrowborder" width="50%">lpx</td>        <td class="cellrowborder" width="50%">视窗逻辑像素单位，lpx单位为实际屏幕宽度与逻辑宽度（通过<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/module-configuration-file#pages标签" target="_blank">designWidth</a>配置）的比值，designWidth默认值为720。当designWidth为720时，在实际宽度为1440物理像素的屏幕上，1lpx为2px大小。</td>       </tr>           </tbody></table></div>    </div>    <div class="tiledSection">
