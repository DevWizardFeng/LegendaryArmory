class="support-device-item">TV</span><span class="support-device-item">Wearable</span></div><p><strong>错误信息</strong></p>     <p>Parameter error.</p>     <p><strong>错误描述</strong></p>     <ol>      <li>       <p>assetKey无效。例如：""（空字符串）。</p></li>      <li>       <p>uri无效。例如：""（空字符串）。</p></li>      <li>       <p>uris无效。例如uris数组长度超过50。</p></li>     </ol>     <p><strong>可能原因</strong></p>     <ol>      <li>       <p>分布式对象中资产类型对应的属性名无效，例如：null（不存在）、undefined（未定义）或''（空字符串）。</p></li>      <li>       <p>待设置的资产需要的分布式路径uri无效，例如：null（不存在）、undefined（未定义）或''（空字符串）。</p></li>      <li>       <p>待设置的资产存放的分布式路径数组为空或数组内元素数量大于50个。</p></li>      <li>       <p>待设置的资产存放的分布式路径数组中存在某个或多个uri无效，例如：null（不存在）、undefined（未定义）或''（空字符串）。</p></li>     </ol>     <p><strong>处理步骤</strong></p>     <ol>      <li>       <p>传入正确的属性名，例如：'file'（文件）。</p></li>      <li>       <p>传入正确的分布式路径uri，例如："file://docs/storage/Users/username/tmp.txt"。</p></li>      <li>       <p>传入正确的分布式路径数组uris，数量为有效范围内的个数，例如：["file://docs/storage/Users/username/tmp.txt","file://com.example.demo/data/storage/el2/base/files/test.txt"]。</p></li>      <li>       <p>确保分布式路径数组uris内每个uri的路径为正确的分布式路径。</p></li>     </ol>    </div>    <div class="tiledSection">     <h2 id="section15400003-已设置分布式对象的sessionid" device-type="phone,2in1,tablet,wearable,tv">15400003
