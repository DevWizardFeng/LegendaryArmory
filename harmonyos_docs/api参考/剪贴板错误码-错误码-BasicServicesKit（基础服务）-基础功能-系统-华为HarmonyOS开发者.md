<p>当调用getRecord等涉及索引的接口时，索引超过范围时，系统会报此错误码。</p>     <p><strong>可能原因</strong></p>     <p>接口的参数索引值超出当前PasteData中的记录数，比如，getRecord传入index值过大。</p>     <p><strong>处理步骤</strong></p>
