<h1 _ngcontent-gmm-c119="" class="doc-title ng-star-inserted" title="多态样式"> 多态样式 </h1>

<div _ngcontent-gmm-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>设置组件不同状态下的样式。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <p>从API version 8开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</p>      <p>从API version 11开始支持另一种写法<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-attribute-modifier">attributeModifier</a>，可根据开发者需要动态设置属性。</p>      <p>多态样式仅支持<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-component-general-attributes">通用属性</a>。如果多态样式不生效，则该属性可能为组件的私有属性，例如：fontColor、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-textinput">TextInput</a>组件的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-background#backgroundcolor18">backgroundColor</a>等。此时，可以通过attributeModifier动态设置组件属性来解决此问题。</p>      <p>当前多态样式实现依赖于组件自定义节点的刷新机制。因Builder不具备独立的自定义父节点，无法直接触发刷新，致使多态样式无法直接在Builder中生效。解决方法是将多态样式封装至自定义组件内部，再将此组件置于@Builder中，以此来间接实现多态样式。示例代码可参考<a href="/consumer/cn/doc/harmonyos-references/ts-universal-attributes-polymorphic-style#示例3设置builder多态样式">示例3设置Builder多态样式</a>。</p>      <p>多态样式的焦点态只有在<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-common-events-focus-event#基础概念" target="_blank">焦点激活态</a>开启时生效。</p>     </div></div></div>    <div class="tiledSection">
