<h1 _ngcontent-puw-c119="" class="doc-title ng-star-inserted" title="工具栏设置"> 工具栏设置 </h1>

<div _ngcontent-puw-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>设置组件对应的工具栏。</p> <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content"> <p> 从API version 20开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</p>  <p> 该toolbar为组件通用属性，请注意与<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation">Navigation</a>组件自身的toolbar属性进行区分。</p>  </div></div></div>  <div class="tiledSection"><h2 id="toolbar" device-type="phone,2in1,tablet,wearable,tv">toolbar<i class="anchor-icon anchor-icon-device active" active-tips="隐藏支持设备" tips="显示支持设备"></i><i class="anchor-icon anchor-icon-link" anchorid="toolbar" tips="复制节点链接"></i></h2><div class="device-list active"><span class="support-device-pre">支持设备</span><span class="support-device-item">Phone</span><span class="support-device-item">PC/2in1</span><span class="support-device-item">Tablet</span><span class="support-device-item">TV</span><span class="support-device-item">Wearable</span></div><p>toolbar(value: CustomBuilder): T</p> <p>为绑定该属性的组件，在窗口顶部标题栏相应分栏创建与该组件绑定的由<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-toolbaritem">ToolBarItem</a>构成的工具栏，分栏位置依据绑定该属性的组件所在分栏位置确定。<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-types#custombuilder8">CustomBuilder</a>必须由<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-toolbaritem">ToolBarItem</a>构成，该工具栏才能生效。</p> <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content"> <p>该接口不支持在<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-attribute-modifier#attributemodifier">attributeModifier</a>中调用。</p>  </div></div></div> class="hw-editor-tip info"><div class="title">说明</div><div class="content"> <ol><li><p>toolbar仅支持固定标题栏，不支持悬浮标题栏（仅限三键模式）。</p>  </li><li><p>toolbar支持自定义组件布局，可将其置于特定分栏位置（左侧或右侧）。但需注意，当元素数量超过可用空间时，将导致布局截断或焦点框遮挡等现象，从而使部分操作项不可见或引发交互冲突。此时，元素不会自动缩略，建议合理控制元素数量。</p>  </li><li><p>toolbar当前仅支持单行布局，不支持多行布局，因此应避免在一个toolbar中放置多行布局的元素。</p>  </li><li><p>toolbar仅支持在<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation#navigationmode9枚举说明">NavigationMode</a>为Split的场景中使用。当<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-navigation#navigationmode9枚举说明">NavigationMode</a>设置为Stack或Auto时，无法应用toolbar。</p>  </li><li><p>标题栏高度会根据toolbar内的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-toolbaritem">ToolBarItem</a>组件在有限范围内浮动：</p>  <ul><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-toolbaritem">ToolBarItem</a>组件与标题栏默认存在4VP的margin（外边距）。</li><li>当<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-toolbaritem">ToolBarItem</a>组件的最大高度小于等于48VP时，标题栏高度会调整为56VP，此设置适用于标题栏、工具栏、搜索栏等通用组件。</li><li>当<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-toolbaritem">ToolBarItem</a>组件的最大高度介于48VP到56VP之间时，标题栏高度会调整为64VP，此设置适用于图标与文字同时呈现的工具栏。</li><li>当<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-toolbaritem">ToolBarItem</a>组件的最大高度超过56VP时，标题栏高度会调整为72VP。如果<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-toolbaritem">ToolBarItem</a>组件的最大高度超过64VP，则标题栏的高度保持为72VP，超出的区域会发生裁剪。</li></ul>  </li></ol>
