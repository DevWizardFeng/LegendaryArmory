class="support-device-item">Wearable</span></div><p>aspectRatio(value: number): T</p> <p>指定当前组件的宽高比，aspectRatio=width/height。</p> <ul><li>仅设置width、aspectRatio时，height=width/aspectRatio。</li><li>仅设置height、aspectRatio时，width=height*aspectRatio。</li><li>同时设置width、height和aspectRatio时，height不生效，height=width/aspectRatio。</li></ul> <p>设置aspectRatio属性后，组件宽高会受父组件内容区大小限制，<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-size#constraintsize">constraintSize</a>的优先级高于aspectRatio。</p>
