      <p><strong>错误信息</strong></p>     <p>Generic failure of print.</p>     <p><strong>可能原因</strong></p>     <p>该错误码表示一般打印异常，可能原因如下：</p>     <ol>      <li>打印内部创建任务对象失败</li>     </ol> anchorid="section13100002-rpc错误" tips="复制节点链接"></i></h2>          <p><strong>错误信息</strong></p>     <p>RPC failure.</p>     <p><strong>可能原因</strong></p>     <p>该错误码表示RPC失败，可能原因如下：</p>     <ol>      <li>获取SA服务对象失败</li>      <li>进程间通信失败</li>     </ol>     <p><strong>处理步骤</strong></p>     <p>请关闭打印界面稍后重试，或重启设备。</p>    </div>    <div class="tiledSection">     <h2 id="section13100003-打印服务异常">13100003 <p><strong>可能原因</strong></p>     <p>该错误码表示打印服务异常，可能原因如下：</p>     <ol>      <li>拉起打印服务失败</li>      <li>拉起cups服务失败</li>     </ol>     <p><strong>处理步骤</strong></p>     <p>请关闭打印界面稍后重试，或重启设备。</p>    </div>    <div class="tiledSection">     <h2 id="section13100004-无效的打印扩展">13100004 无效的打印扩展<i class="anchor-icon anchor-icon-link" anchorid="section13100004-无效的打印扩展" tips="复制节点链接"></i></h2>          <p><strong>错误信息</strong></p>     <p>Invalid print extension.</p>     <p><strong>可能原因</strong></p>     <p>该错误码表示账号已存在，可能原因如下：</p>     <ol>      <li>找不到打印扩展</li>      <li>打印扩展参数错误</li>      <li>打印扩展状态错误</li>     </ol>     <p><strong>处理步骤</strong></p>     <p>请关闭打印界面稍后重试，或重启设备。</p>    </div>    <div class="tiledSection">     <h2 id="section13100005-无效的打印机">13100005 无效的打印机<i <p>该错误码表示不支持多用户，可能原因如下：</p>     <ol>      <li>打印机id错误</li>      <li>cups查询打印机列表失败</li>      <li>cups查询打印机信息失败</li>     </ol>     <p><strong>处理步骤</strong></p>     <p>请关闭打印界面稍后重试，或重启设备。</p>    </div>    <div class="tiledSection">     <h2 id="section13100006-无效的打印任务">13100006 无效的打印任务<i class="anchor-icon anchor-icon-link" anchorid="section13100006-无效的打印任务" tips="复制节点链接"></i></h2>          <p><strong>错误信息</strong></p>     <p>Invalid print job.</p>     <p><strong>可能原因</strong></p>     <p>该错误码表示提供了不支持的账号类型，可能原因如下：</p>     <ol>      <li>打印任务id错误</li>      <li>打印任务状态错误</li>      <li>找不到打印任务</li>     </ol>     <p><strong>处理步骤</strong></p>     <p>请取消任务或关闭打印界面重试，或重启设备。</p>    </div>   </div>   <div></div></div>
