   <li>       <p>引用资源文件，推荐使用绝对路径。比如：/common/test.png。</p></li>      <li>       <p>公共代码文件和资源文件推荐放在common下，通过规则1和规则2进行访问。</p></li>      <li>       <p>CSS样式文件中通过url()函数创建&lt;url&gt;数据类型，如：url(/common/test.png)。</p></li>     </ul>     <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">       <p>当代码文件A需要引用代码文件B时：</p>       <ul>        <li>         <p>如果代码文件A和文件B位于同一目录，则代码文件B引用资源文件时可使用相对路径，也可使用绝对路径。</p></li>        <li>         <p>如果代码文件A和文件B位于不同目录，则代码文件B引用资源文件时必须使用绝对路径。因为Webpack打包时，代码文件B的目录会发生变化。</p></li>        <li>         <p>在json文件中定义的数据为资源文件路径时，需使用绝对路径。</p></li>       </ul>      </div></div></div>    </div>    <div
