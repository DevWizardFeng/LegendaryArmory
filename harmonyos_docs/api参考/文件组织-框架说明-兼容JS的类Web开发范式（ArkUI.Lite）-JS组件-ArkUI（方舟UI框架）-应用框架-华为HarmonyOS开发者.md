<ul><li><p>引用代码文件，需使用相对路径，比如：../common/utils.js。</p>  </li><li><p>引用资源文件，推荐使用绝对路径。比如：/common/xxx.png。</p>  </li><li><p>公共代码文件和资源文件推荐放在common下，通过以上两条规则进行访问。</p>  </li><li><p>CSS样式文件中通过url()函数创建&lt;url&gt;数据类型，如：url(/common/xxx.png)。</p>  </li></ul> <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content"> <p> 当代码文件A需要引用代码文件B时：</p>
