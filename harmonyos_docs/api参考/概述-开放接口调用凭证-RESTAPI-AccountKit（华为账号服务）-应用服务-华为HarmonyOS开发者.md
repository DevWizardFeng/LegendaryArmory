<div _ngcontent-von-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>在华为账号开放接口中，凭证是调用各类接口的核心鉴权凭据，贯穿身份认证、权限校验、数据安全等关键环节。</p> <div class="tiledSection"><h2 id="section1167014476118">场景介绍<i class="anchor-icon anchor-icon-link" anchorid="section1167014476118" tips="复制节点链接"></i></h2><ul><li><strong>获取用户级凭证</strong><p>应用通过用户登录、授权获取的授权码（Authorization Code），获取用户级凭证Access Token，然后通过该凭证Access Token访问账号受权限管控的接口（如<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/account-api-get-user-info-get-nickname-and-avatar">获取华为账号用户信息-获取头像昵称</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/account-api-getuserrisklevel">获取用户风险等级</a>等）。</p> </li><li><strong>刷新用户级凭证</strong><p>提供用户级凭证Access Token主动刷新机制，应用可通过定期刷新凭证信息避免接口调用中断。</p> </li><li><strong>解析凭证</strong><p>提供对凭证的解析能力，应用可通过该场景获取凭证携带的关键信息（如应用Client ID、用户身份、有效期）。</p> </li><li><strong>取消用户级凭证授权</strong><p>当用户主动解除授权或应用需要终止服务时，可通过该场景立即废弃凭证。</p> </li><li><strong>获取应用级凭证</strong><p>通过应用的Client ID和Client Secret获取应用级凭证Access Token，用于调用应用级权限管控接口（如<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/account-api-get-unionid">通过OpenID获取UnionID</a>）。</p> </li></ul> </div> </div> <div></div></div>
