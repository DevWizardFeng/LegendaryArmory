<h1 _ngcontent-lxk-c119="" class="doc-title ng-star-inserted" title="模块描述"> 模块描述 </h1>

<div _ngcontent-lxk-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>本模块为开发者提供了一套简单易懂的相机服务接口。通过调用这些接口，开发者可以开发相机应用，访问和操作相机硬件，实现基础功能如预览、拍照和录像。此外，还可以通过接口组合完成更多操作，如控制闪光灯、曝光时间和对焦等。</p> <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content"> <p>本模块首批接口从API version 10开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。</p>  </div></div></div> <p>本模块包含以下基础类：</p> <ul><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-autodeviceswitch">AutoDeviceSwitch</a>类，用于查询设备是否支持自动切换镜头，使能或去使能自动切换镜头。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-autoexposure">AutoExposure</a>类，针对设备的自动曝光特性提供了一系列查询功能，支持设置或查询当前曝光值、曝光区域中心点以及曝光补偿。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-camerainput">CameraInput</a>类，提供开关相机，以及监听输入流异常的能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-cameramanager">CameraManager</a>类，提供获取相机设备对象、创建输入/输出流以及会话Session、注册/解注册监听相机相关状态等能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-colormanagement">ColorManagement</a>类，提供查询是否支持色彩空间，设置以及获取色彩空间参数的能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-controlcenter">ControlCenter</a>类，提供查询是否支持相机控制器，以及使能相机控制器的能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-flash">Flash</a>类，提供查询是否支持闪光灯功能，以及具有对闪光灯设备操作的能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-focus">Focus</a>类，提供查询对焦模式在设备上是否支持的接口，以及设置对焦相关功能的能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-macro">Macro</a>类，提供查询是否支持微距，以及使能微距的能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-metadataoutput">MetadataOutput</a>类，用于注册/解注册监听meta数据以及启动/停止meta流。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-photo">Photo</a>类，全质量图对象，包含一张图片的完整信息。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-photooutput">PhotoOutput</a>类，拍照会话中使用的输出信息，提供拍照相关的能力，如拍照、使能动态拍照和获取拍照旋转角度等。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-photosession">PhotoSession</a>类，普通拍照模式会话类，提供了预配置以及会话状态监听相关的能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-previewoutput">PreviewOutput</a>类，提供对预览流帧率、预览旋转角度和查询当前配置信息的能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-securesession">SecureSession</a>类，安全模式会话类，提供是否支持安全模式查询以及会话状态监听的相关能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-session">Session</a>类，会话的基础类，提供了预览流启流到停流的一系列接口。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-stabilization">Stabilization</a>类，提供查询设备是否支持防抖，以及获取/设置设备防抖类型的能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-videooutput">VideoOutput</a>类，提供启动/停止录制流，以及配置录制流帧率、镜像等能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-videosession">VideoSession</a>类，普通录像模式会话类，提供了预配置以及监听会话状态相关的能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-whitebalance">WhiteBalance</a>类，提供检测是否支持白平衡模式以及设置白平衡相关参数的能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-zoom">Zoom</a>类，提供查询是否支持变焦操作以及对设备进行变焦操作的能力。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-i">其他</a>：相机设备的属性信息，包含视频配置项、拍照输出能力和手电筒等。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-camera-e">Enums</a>：相机相关的枚举及其含义。包含相机位置、闪光灯模式和平滑变焦模式等。</li></ul>  <div class="tiledSection"><h2 id="导入模块"
