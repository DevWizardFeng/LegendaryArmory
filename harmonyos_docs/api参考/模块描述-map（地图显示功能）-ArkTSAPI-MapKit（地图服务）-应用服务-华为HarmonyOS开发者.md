<div _ngcontent-utu-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>map（地图显示功能）为开发者提供易于上手的接口，开发者可以通过相关接口便捷地在HarmonyOS应用/元服务中加入地图相关的功能，包括显示地图、在地图上绘制（覆盖物）、添加动画、与地图交互、更新地图状态、常用工具函数等功能。</p> <p>该模块提供以下地图常用功能：</p> <ul><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-mapcomponentcontroller">MapComponentController</a>：显示地图，与地图有关的所有方法从此处接入。</li></ul> <p>地图覆盖物：</p> <ul><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-marker">Marker</a>：标记。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-mappolyline">MapPolyline</a>：折线。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-maparc">MapArc</a>：弧线。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-mappolygon">MapPolygon</a>：多边形。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-mapcircle">MapCircle</a>：圆形。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-pointannotation">PointAnnotation</a>：点注释。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-bubble">Bubble</a>：气泡。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-clusteroverlay">ClusterOverlay</a>：点聚合。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-imageoverlay">ImageOverlay</a>：图片覆盖物。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-buildingoverlay">BuildingOverlay</a>：3D建筑。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-traceoverlay">TraceOverlay</a>：动态轨迹。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-tileoverlay">TileOverlay</a>：瓦片图层。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-heatmap">Heatmap</a>：热力图。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-mvtoverlay">MvtOverlay</a>：矢量图层。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-flowfieldoverlay">FlowFieldOverlay</a>：流场图层。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-masspointoverlay">MassPointOverlay</a>：海量点图层。</li></ul> <p>添加动画：</p> <ul><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-animation">Animation</a>：动画抽象类。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-alphaanimation">AlphaAnimation</a>：控制透明度的动画类。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-rotateanimation">RotateAnimation</a>：控制旋转的动画类。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-scaleanimation">ScaleAnimation</a>：控制缩放的动画类。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-translateanimation">TranslateAnimation</a>：控制移动的动画类。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-fontsizeanimation">FontSizeAnimation</a>：控制字体大小的动画类。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-playimageanimation">PlayImageAnimation</a>：控制多张图片的帧动画类。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-animationset">AnimationSet</a>：动画类集合。</li></ul> <p>与地图交互：</p> <ul><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-mapeventmanager">MapEventManager</a>：地图监听事件管理器。</li></ul> <p>更新地图状态：</p> <ul><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-newlatlng">newLatLng</a>：设置地图的中心点和缩放层级。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-newlatlngbounds-1">newLatLngBounds</a>：设置地图经纬度范围、地图区域和边界之间的距离。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-newlatlngbounds-2">newLatLngBounds</a>：设置地图经纬度范围、经纬度矩形范围的高和宽、地图区域和边界之间的距离。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-newlatlngbounds-3">newLatLngBounds</a>：设置地图经纬度范围和4个方向与边界之间的距离。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-scrollby">scrollBy</a>：按像素移动地图中心点。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-zoomby">zoomBy</a>：根据给定增量并以给定的屏幕像素点为中心点缩放地图级别。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-zoomin">zoomIn</a>：放大地图缩放级别，在当前地图显示的级别基础上加1。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-zoomout">zoomOut</a>：缩小地图缩放级别，在当前地图显示的级别基础上减1。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-zoomto">zoomTo</a>：设置地图缩放级别。</li></ul> <p>常用工具函数：</p> <ul><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-calculatedistance">calculateDistance</a>：计算坐标点之间的距离。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-convertcoordinate">convertCoordinate</a>：坐标系转换，使用Promise异步回调。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-convertcoordinatesync">convertCoordinateSync</a>：坐标系转换。</li><li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/map-map-rectifycoordinate">rectifyCoordinate</a>：根据用户输入的坐标系和坐标以及获取当前的路由地，判断是否需要修正坐标。</li></ul> <div class="tiledSection"><h2
