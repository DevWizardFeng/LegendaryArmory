tips="复制节点链接"></i></h2>          <p><strong>错误信息</strong></p>     <p>The node type is not custom node.</p>     <p><strong>错误描述</strong></p>     <p>当前操作的节点不是NDK侧的自定义节点。</p>     <p><strong>可能原因</strong></p>     <p>开发者传入的ArkUI_NodeHandle指针所对应的节点类型非ARKUI_NODE_CUSTOM类型。</p>     <p><strong>处理步骤</strong></p>     <p>接入渲染节点流程时，创建一个Custom类型的NDK节点作为渲染节点的根节点类型。</p>    </div>    <div class="tiledSection">     <h2 id="section106402-当前节点已存在子节点">106402 当前节点已存在子节点<i class="anchor-icon anchor-icon-link" anchorid="section106402-当前节点已存在子节点" tips="复制节点链接"></i></h2>          <p><strong>错误信息</strong></p>     <p>Node already has children.</p>     <p><strong>错误描述</strong></p>     <p>当前操作的自定义节点已存在子节点。</p>     <p><strong>可能原因</strong></p>     <p>开发者构造render树时，作为根节点的自定义节点已挂载子FrameNode或RenderNode。</p>     <p><strong>处理步骤</strong></p>     <p>接入渲染节点流程时，排查使用的自定义节点是否已存在子节点。</p>    </div>    <div class="tiledSection">     <h2 <p>RenderNode parent is existed.</p>     <p><strong>错误描述</strong></p>     <p>当前操作的渲染节点已有父节点，无法挂载至相应自定义组件下。</p>     <p><strong>可能原因</strong></p>     <p>开发者传入的ArkUI_RenderNodeHandle指针所对应的节点已挂载至其他组件下。</p>     <p><strong>处理步骤</strong></p>     <p>接入渲染节点流程时，排查作为根节点的renderNode是否已挂载至其他组件下。</p>    </div>    <div class="tiledSection">     <h2 id="section106404-未找到对应的渲染子节点">106404 未找到对应的渲染子节点<i class="anchor-icon anchor-icon-link" anchorid="section106404-未找到对应的渲染子节点" tips="复制节点链接"></i></h2>          <p><strong>错误信息</strong></p>     <p>RenderNode child is not exist.</p>     <p><strong>错误描述</strong></p>     <p>无法查找到对应的渲染子节点。</p>     <p><strong>可能原因</strong></p>     <p>开发者传入的ArkUI_RenderNodeHandle指针所对应的渲染节点无法查询到对应下标的子节点。</p>     <p><strong>处理步骤</strong></p>     <p>排查传入的下标是否超出节点范围，或对应指针对应的渲染节点是否存在子节点。</p>    </div>    <div class="tiledSection">     <h2 id="section106405-参数值超出范围">106405 参数值超出范围<i class="anchor-icon <p>入参值超出接口接收值的范围。</p>     <p><strong>可能原因</strong></p>     <p>开发者传入的参数超出该接口的边界。</p>     <p><strong>处理步骤</strong></p>     <p>检查接口调用的入参范围。</p>    </div>   </div>   <div></div></div>
