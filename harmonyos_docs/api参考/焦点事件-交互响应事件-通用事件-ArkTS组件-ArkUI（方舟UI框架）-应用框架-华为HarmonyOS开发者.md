<h1 _ngcontent-nem-c119="" class="doc-title ng-star-inserted" title="焦点事件"> 焦点事件 </h1>

<div _ngcontent-nem-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>焦点事件指页面焦点在可获焦组件间移动时触发的事件，组件可使用焦点事件来处理相关逻辑。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <ul>       <li>        <p>本模块首批接口从API version 8开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。</p></li>       <li>        <p>不支持嵌套滚动组件场景按键走焦。</p></li>       <li>        <p>存在默认交互逻辑的组件例如<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-button">Button</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-textinput">TextInput</a>等，默认即为可获焦，<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-text">Text</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-image">Image</a>等组件默认状态为不可获焦，不可获焦状态下，无法触发焦点事件，需要设置<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-focus#focusable">focusable</a>属性为true才可触发。</p></li>       <li>        <p>对于有获焦能力的容器组件，例如<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-stack">Stack</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-row">Row</a>等，若不存在可获焦子组件，该容器组件不可获焦。为其配置onClick或是单指单击的Tap手势，且不显式配置focusable属性，该组件会隐式地成为可获焦组件。</p></li>       <li>        <p>焦点开发及组件获焦能力参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-common-events-focus-event" target="_blank">焦点开发指南</a>。</p></li>      </ul>     </div></div></div>
