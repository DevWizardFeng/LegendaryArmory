<h1 _ngcontent-drf-c119="" class="doc-title ng-star-inserted" title="组件区域变化事件"> 组件区域变化事件 </h1>

<div _ngcontent-drf-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>组件区域变化事件指组件显示的尺寸、位置等发生变化时触发的事件。</p> <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content"> <p> 从API version 8开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</p>  <p>onAreaChange回调执行仅与本组件有关，对祖先或子孙组件上的onAreaChange的回调没有严格的执行顺序和限制保证。</p> <p>由绘制变化所导致的渲染属性变化不会响应回调，如<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-transformation#translate">translate</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-transformation#transform">transform</a>。若组件自身位置由绘制变化决定也不会响应回调，如<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-sheet-transition#bindsheet">bindSheet</a>。</p> <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content"> <p>当组件同时绑定onAreaChange事件和<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-types#position">Position</a>类型的position属性变化，不响应设置<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-types#edges12">Edges</a>和<a
