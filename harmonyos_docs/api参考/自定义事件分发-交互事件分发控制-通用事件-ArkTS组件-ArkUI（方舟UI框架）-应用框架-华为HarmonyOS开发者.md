<h1 _ngcontent-bld-c119="" class="doc-title ng-star-inserted" title="自定义事件分发"> 自定义事件分发 </h1>

<p>onClick和旋转、捏合手势经过自定义事件分发后，可能会因为未命中触摸热区导致事件不响应。</p></li>      </ul>     </div></div></div>    <div class="tiledSection">     <h2 id="onchildtouchtest11"
