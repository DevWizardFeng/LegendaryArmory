<h1 _ngcontent-kga-c119="" class="doc-title ng-star-inserted" title="触摸热区设置"> 触摸热区设置 </h1>

<div _ngcontent-kga-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>设置组件的响应热区。在ArkUI开发框架中，处理触屏事件和鼠标事件时，会在事件触发前进行按压点与组件响应热区的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-interaction-basic-principles#触摸测试" target="_blank">触摸测试</a>，以收集需响应事件的组件。基于测试结果，框架会分发相应的事件。影响<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-events-click">点击事件</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-events-touch">触摸事件</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-events-drag-drop">拖拽事件</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-mouse-key">鼠标事件</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-events-axis">轴事件</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-events-hover">悬浮事件</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-accessibility-hover-event">无障碍悬浮事件</a>和<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-gesture-settings">手势事件</a>的分发。</p>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <ul>       <li>从API version 8开始支持。后续版本如有新增内容，则采用上角标单独标记该内容的起始版本。</li>       <li>设置触摸热区属性时，手指需在热区内按下，随后抬起时若满足事件响应条件，事件将被触发。此外，在当前手势结束前，若条件满足，可持续触发的事件也会被激活。</li>      </ul>     </div></div></div> <p>x和y可以设置正负值百分比。当x设置为'100%'时表示热区往右偏移组件本身宽度大小，当x设置为'-100%'时表示热区往左偏移组件本身宽度大小。当y设置为'100%'时表示热区往下偏移组件本身高度大小，当y设置为'-100%'时表示热区往上偏移组件本身高度大小。</p>       <p>width和height只能设置正值百分比。width：'100%'表示热区宽度设置为该组件本身的宽度。比如组件本身宽度是100vp，那么'100%'表示热区宽度也为100vp。height：'100%'表示热区高度设置为该组件本身的高度。</p>       <p>百分比相对于组件自身宽高进行计算。</p>       <p>当父组件设置<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-sharp-clipping#clip12">clip</a>(true)时，子组件的响应会受到父组件触摸热区的影响，不在父组件触摸热区内的子组件无法响应手势和事件。</p>       <p>width和height不支持calc()的动态计算。</p>      </div></div></div>    </div>    <div class="tiledSection">     <h2 id="示例" device-type="phone,2in1,tablet,wearable,tv">示例<i class="anchor-icon
