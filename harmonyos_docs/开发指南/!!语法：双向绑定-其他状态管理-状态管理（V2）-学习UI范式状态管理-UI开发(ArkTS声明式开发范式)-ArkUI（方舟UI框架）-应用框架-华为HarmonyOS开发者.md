<div _ngcontent-ptx-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>在状态管理V1中，推荐使用<a anchorid="概述" tips="复制节点链接"></i></h2>          <p>!!双向绑定语法，是一个语法糖方便开发者实现数据双向绑定，用于初始化子组件的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-new-param">@Param</a>和<a href="/consumer/cn/doc/harmonyos-guides/arkts-new-binding#使用场景">使用场景</a>。</p>     <ul>      <li>如果使用了!!双向绑定语法，表明父组件的变化会同步给子组件，子组件的变化也会同步给父组件。</li>      <li>父组件未使用!!时，变化是单向的。</li>     </ul>    </div>
