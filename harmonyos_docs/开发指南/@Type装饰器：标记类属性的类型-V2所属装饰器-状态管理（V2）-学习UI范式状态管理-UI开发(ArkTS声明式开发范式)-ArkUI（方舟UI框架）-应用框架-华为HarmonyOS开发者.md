<div _ngcontent-dlw-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>为了实现序列化类时不丢失属性的复杂类型，开发者可以使用@Type装饰器装饰类属性。</p>    <p>@Type的目的是标记类属性，配合PersistenceV2使用，防止序列化时类丢失。在阅读本文档前，建议提前阅读：<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-new-persistencev2">PersistenceV2</a>。</p>    <div><div
