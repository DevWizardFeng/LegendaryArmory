<h1 _ngcontent-qrg-c119="" class="doc-title ng-star-inserted" title="App Linking Kit简介"> App Linking Kit简介 </h1>

<div _ngcontent-qrg-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>App Linking Kit（应用链接服务）提供了一系列增强的链接特性。</p>    <ul>     <li>App Linking Kit支持<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/app-linking-startupapp">通过App Linking应用链接拉起指定应用</a>，实现应用间跳转。当应用已安装时，优先通过应用展示内容；若应用未安装，则通过系统浏览器展示网页版内容。       <div class="p">       在此基础之上，还可以实现延迟链接能力、直达应用市场能力这类有竞争力的特性，大大增强了App Linking的能力，使得链接跳转体验更佳，链接转化率更高。        <ul>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/applinking-deferredlink">通过延迟链接跳转至应用详情页</a>：当用户点击应用推广链接时，若应用未安装，系统会将用户的点击信息自动缓存十分钟。当用户随后安装并启动应用时，仍可获取之前的点击参数，避免转化率损失，提升体验。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/applinking-direct-to-ag">通过直达应用市场能力跳转至应用市场下载详情页</a>：当应用未安装时，App Linking的默认行为是通过系统浏览器打开链接对应的网页。通过App Linking Kit的直达应用市场能力，可以实现在应用未安装时直接跳转应用市场，省去了中转的步骤，使跳转体验更流畅。</li>       </ul>      </div></li>     <li>App Linking Kit支持<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/applinking-cross-platform">通过聚合链接按指定方式跳转至应用</a>。当用户在HarmonyOS系统中点击聚合链接时，默认通过系统浏览器打开深度链接地址。通过聚合链接能力，可以引导用户跳转到HarmonyOS平台预览页、应用市场详情页、自定义网址、深度链接地址等页面。</li>    </ul>    <div class="tiledSection">     <h2 id="section77453315211">适用场景<i class="anchor-icon anchor-icon-link" anchorid="section77453315211" tips="复制节点链接"></i></h2>          <ul>      <li>适用于应用的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/scan-directservice" target="_blank">扫码直达</a>、社交分享、沉默唤醒、广告引流等场景。</li>      <li>适用于对安全性要求较高的场景，避免出现被其它应用仿冒的问题。</li>      <li>适用于对体验要求较高的应用，不管目标应用是否安装，用户点击该链接都可以正常访问。</li>     </ul>    </div>    <div class="tiledSection">     <h2 id="section43285313545">典型案例<i class="anchor-icon anchor-icon-link" anchorid="section43285313545" tips="复制节点链接"></i></h2>         </div>    <div class="tiledSection">     <h3 id="section11232629468" class="firsth2">碰一碰视频分享<i class="anchor-icon anchor-icon-link" anchorid="section11232629468" tips="复制节点链接"></i></h3>          <p>随着全场景智慧生活的不断演进，跨设备内容分享已成为用户的核心需求之一。传统分享方式普遍存在操作繁琐（需手动选择设备或应用）、依赖特定网络环境、传输效率低等问题，影响了用户体验。HarmonyOS提供的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/share-introduction" target="_blank">Share Kit（分享服务）</a>结合App Linking Kit技术，能够实现内容的快速跨设备分享，直达目标应用，无需依赖第三方应用中转，提供高效、便捷、无缝的分享体验。</p>
