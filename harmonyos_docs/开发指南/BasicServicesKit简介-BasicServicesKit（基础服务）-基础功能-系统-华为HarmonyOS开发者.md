<h1 _ngcontent-rxj-c119="" class="doc-title ng-star-inserted" title="Basic Services Kit简介"> Basic Services Kit简介 </h1>

<div _ngcontent-rxj-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>Basic Services Kit（基础服务）作为基础服务套件，为应用开发者提供常用的基础能力。比如常用的剪贴板读写、文件上传下载、文件压缩、文件打印、进程间/线程间通信、设备管理、应用账号管理等能力都由本Kit提供。</p>    <div class="tiledSection">     <h2 id="使用场景">使用场景<i class="anchor-icon anchor-icon-link" anchorid="使用场景" tips="复制节点链接"></i></h2>          <p>Basic Services Kit为开发者提供了多种基础能力，满足开发者不同场景的开发需求。</p>     <p>典型使用场景举例：</p>     <ul>      <li>       <p>剪贴板读写：</p>       <ul>        <li>本地复制粘贴：比如在A应用中复制一段文字，粘贴到其他应用中。</li>        <li>跨设备复制粘贴：比如在A设备浏览器复制一段文本，粘贴到B设备的备忘录中。</li>       </ul></li>      <li>       <p>文件上传下载：</p>       <ul>        <li>小文件前台上传下载：发布社交动态（图片、短视频等）、发送文件给好友、保存图片到本地等，通常数据量较小，要给用户即时反馈。</li>        <li>大文件后台上传下载：云盘数据同步、下载电影，通常数据量较大，可后台执行，支持断点续传。</li>       </ul></li>      <li>       <p>进程间/线程间通信：</p>       <ul>        <li>进程间通信：比如ExtensionAbility发送事件给主进程。</li>        <li>线程间通信：比如worker线程处理完网络请求后将事件传递回UI主线程。</li>       </ul></li>     </ul>    </div>    <div class="tiledSection">     <h2 id="能力范围">能力范围<i class="anchor-icon anchor-icon-link" anchorid="能力范围" tips="复制节点链接"></i></h2>          <p>根据不同使用场景分类，本Kit主要包含如下能力：</p>     <ul>      <li>       <p>数据文件处理：</p>       <ul>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-pasteboard" target="_blank">剪贴板</a>：提供内容复制粘贴能力，支持多种数据类型包括文本、HTML数据、URI、PixelMap等。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-zlib" target="_blank">压缩</a>：提供文件压缩解压缩的能力。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-print" target="_blank">打印</a>：提供基础文件打印的能力，比如传入文件进行打印、设置打印参数等。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-request" target="_blank">上传下载</a>：提供文件上传下载、后台传输代理的基础能力。</li>       </ul></li>      <li>       <p>进程间/线程间通信：</p>       <ul>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-commoneventmanager" target="_blank">公共事件</a>：提供进程间通信的能力，包括订阅、发布、退订公共事件等，相关开发指南请参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/common-event-overview">公共事件简介</a>。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-emitter" target="_blank">Emitter</a>：提供线程内通信的能力，包括订阅、发布、退订自定义事件等，相关开发指南请参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/itc-with-emitter">使用Emitter进行线程间通信</a>。</li>       </ul></li>      <li>       <p>设备管理：</p>       <ul>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-device-info" target="_blank">设备信息</a>：提供查询产品信息的能力，比如查询设备类型、设备品牌名称、产品系列、产品版本号等。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-settings" target="_blank">设置数据项</a>：提供查询系统设置数据项的能力，比如查询是否启用飞行模式、是否启用触摸浏览等。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-battery-info" target="_blank">电量信息查询</a>：提供查询电量信息的能力。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-power" target="_blank">系统电源管理</a>：提供系统电源管理相关的能力，比如查询屏幕状态能力等。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-runninglock" target="_blank">RunningLock锁操作</a>：提供RunningLock锁相关操作的能力，包括创建、查询、持锁、释放锁等操作。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-thermal" target="_blank">热管理</a>：提供热管理相关的能力，比如热档位查询等。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-usbmanager" target="_blank">USB管理</a>：提供USB设备管理相关的能力，比如查询USB设备列表、批量数据传输、控制命令传输、权限控制等，相关开发指南请参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/usbhost-overview">USB服务开发概述</a>。</li>       </ul></li>      <li>       <p>其他：</p>       <ul>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-appaccount" target="_blank">应用账号管理</a>：提供应用账号的期管理以及数据管理的能力，相关开发指南请参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/manage-application-account">管理应用账号</a>。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-base" target="_blank">公共回调</a>：定义了HarmonyOS ArkTS接口的公共回调类型，包括接口调用时出现的公共回调和公共错误信息。</li>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-date-time" target="_blank">时间时区</a>：提供获取系统时间以及系统时区的能力。</li>       </ul></li>     </ul>    </div>    <div class="tiledSection">     <h2 id="与其他kit的关系">与其他kit的关系<i class="anchor-icon anchor-icon-link" anchorid="与其他kit的关系" tips="复制节点链接"></i></h2>          <ul>      <li>       <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-overview">ArkTS Kit</a>：ArkTS Kit中的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/multi-thread-concurrency-overview">多线程并发(Worker和Taskpool)</a>需要使用本Kit中的Emitter进行线程间通信。</p></li>      <li>       <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/abilitykit-overview">Ability Kit</a>：Ability Kit中的进程间通信需要使用本Kit中的公共事件。</p></li>      <li>       <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/core-file-kit-intro">Core File Kit</a>：与Core File Kit的使用场景不同，Core File Kit主要提供访问和管理文件的能力，开发者可以在应用文件访问和文件分享、应用数据备份恢复等场景使用Core File Kit进行开发，而涉及到文件压缩、文件上传下载、文件打印等场景时需要使用本Kit进行开发。</p></li>     </ul>    </div>   </div>   <div></div></div>
