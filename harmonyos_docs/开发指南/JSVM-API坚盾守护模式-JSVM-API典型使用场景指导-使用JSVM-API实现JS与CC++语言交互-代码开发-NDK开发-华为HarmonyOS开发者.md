<div _ngcontent-wxa-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>坚盾守护模式是为高安全需求用户设计的系统级安全防护方案。该模式通过实施严格的功能限制，显著增强系统安全性，有效防范针对远程攻击面的各类威胁。在坚盾守护模式下，JSVM 增加了功能限制，需要开发者评估应用在坚盾守护模式下的可用性。</p>  <div class="tiledSection"><h2 id="启用坚盾守护模式">启用坚盾守护模式<i class="anchor-icon anchor-icon-link" anchorid="启用坚盾守护模式" tips="复制节点链接"></i></h2><p>要启用坚盾守护模式，请按以下路径操作：</p> <ol><li>进入手机系统设置</li><li>选择"隐私和安全"选项</li><li>点击"坚盾守护模式"并开启</li></ol> </div>  <div class="tiledSection"><h2 id="坚盾守护模式下的功能限制">坚盾守护模式下的功能限制<i class="anchor-icon anchor-icon-link" anchorid="坚盾守护模式下的功能限制" tips="复制节点链接"></i></h2><p>为降低JSVM受攻击风险，坚盾守护模式将实施以下关键安全限制：</p> <ul><li>全面禁用即时编译(JIT)功能，包括已获取 ACL 权限的应用程序</li><li>暂停 WebAssembly 支持（当前版本中 WebAssembly 依赖 JIT 功能实现）</li></ul> </div>  <div class="tiledSection"><h2 id="应用兼容性评估指南">应用兼容性评估指南<i class="anchor-icon anchor-icon-link" anchorid="应用兼容性评估指南" tips="复制节点链接"></i></h2><p>在坚盾守护模式下运行应用程序时，建议进行以下兼容性检查：</p> <ol><li><p>JavaScript 性能评估：</p>  <ul><li>测试应用在限制环境中的运行效率</li><li>优化可能存在的性能瓶颈</li></ul>  </li><li><p>WebAssembly 兼容性检查：</p>  <ul><li>静态代码分析：检查项目中的 WebAssembly 相关API调用，与第三方库的 Wasm 依赖情况。</li><li>运行时验证：在坚盾守护模式下执行全功能测试。</li></ul>  </li></ol> </div> </div> <div></div></div>
