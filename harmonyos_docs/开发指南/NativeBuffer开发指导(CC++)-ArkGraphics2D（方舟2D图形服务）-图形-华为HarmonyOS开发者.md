class="anchor-icon anchor-icon-link" anchorid="场景介绍" tips="复制节点链接"></i></h2>          <p>NativeBuffer是提供<strong>共享内存</strong>的模块。开发者可以通过NativeBuffer接口实现共享内存的申请、使用、属性查询、释放等操作。</p>     <p>针对NativeBuffer，常见的开发场景如下：</p>     <ul>      <li>通过NativeBuffer提供的Native API接口申请OH_NativeBuffer实例，获取内存的属性信息，把对应的ION内存映射到进程空间。</li>     </ul>    </div>    <div class="tiledSection">     <h2
