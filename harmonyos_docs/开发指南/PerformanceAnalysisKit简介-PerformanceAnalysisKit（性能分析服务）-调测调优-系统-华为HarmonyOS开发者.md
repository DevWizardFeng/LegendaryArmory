<h1 _ngcontent-hex-c119="" class="doc-title ng-star-inserted" title="Performance Analysis Kit简介"> Performance Analysis Kit简介 </h1>

<div _ngcontent-hex-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>Performance Analysis Kit（性能分析服务）为开发者提供应用事件、日志、跟踪分析工具，可观测应用运行时状态，用于行为分析、故障分析、安全分析、统计分析，帮助开发者持续改进应用体验。</p> <p>Performance Analysis Kit承载着HarmonyOS DFX子系统面向应用开发者提供的提升应用质量能力集合，详见<a href="https://developer.huawei.com/consumer/cn/doc/best-practices/bpta-quality-overview" target="_blank">应用质量概览</a>。</p>  <div class="tiledSection"><h2 id="使用场景">使用场景<i class="anchor-icon anchor-icon-link" anchorid="使用场景" tips="复制节点链接"></i></h2><ul><li><p>应用调试：提供流水日志功能，帮助开发者记录和获取日志，以便进行问题分析。</p>  </li><li><p>问题定位：提供各类场景的故障现场，包含可靠性、性能、功耗、分布式故障相关的日志、事件、跟踪。</p>  </li><li><p>线上监控：提供应用构建线上观测所需的日志、事件、跟踪接口，方便开发者记录和分析应用的线上运行情况。</p>  </li></ul> </div>  <div class="tiledSection"><h2 id="能力范围">能力范围<i class="anchor-icon anchor-icon-link" anchorid="能力范围" </li><li><p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/power-detection" target="_blank">功耗检测</a>，提供开发者检测应用功耗异常的能力，如CPU高负载检测。</p>  </li><li><p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/hichecker-guidelines-arkts">检测模式</a>，提供开发者检测应用线程耗时调用、元能力资源泄漏等问题的能力。</p>  </li><li><p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/hidebug-guidelines">系统调试信息获取</a>，提供开发者获取应用和系统资源使用情况的能力。</p>  </li><li><p><a
