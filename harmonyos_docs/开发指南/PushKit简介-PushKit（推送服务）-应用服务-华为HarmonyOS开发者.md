<h1 _ngcontent-sic-c119="" class="doc-title ng-star-inserted" title="Push Kit简介"> Push Kit简介 </h1>

<div _ngcontent-sic-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>Push Kit（推送服务）是华为提供的消息推送平台，建立了从云端到终端的消息推送通道。所有HarmonyOS应用可通过集成Push Kit，实现向应用实时推送消息，使消息易见，构筑良好的用户关系，提升用户的感知度和活跃度。</p>    <div class="tiledSection">     <h2 id="section1965651143017">快速入门<i class="anchor-icon anchor-icon-link" anchorid="section1965651143017" tips="复制节点链接"></i></h2>          <p>请参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/push-gettingstart">使用入门</a>章节快速了解接入Push Kit（推送服务）的必要步骤。</p>    </div>    <div class="tiledSection">     <h2 id="section55411333183020">产品优势<i class="anchor-icon anchor-icon-link" anchorid="section55411333183020" tips="复制节点链接"></i></h2>          <ul>      <li><strong>稳定的消息发送通道</strong>       <p>Push Kit通过提供系统级长链接，即使应用进程不在也能实时推送消息。</p></li>      <li><strong>丰富的消息呈现样式</strong>       <p>支持文本样式、通知大图标样式、多行文本样式、角标样式等多种消息展示方式，满足您多样化、个性化的消息发送需求。</p></li>      <li><strong>灵活的场景化消息</strong>       <p>开发者可以根据实际场景灵活接入场景化消息。如通过应用内通话消息实现音视频通话，通过通知扩展消息实现语音播报业务处理，通过后台消息实现配置更新等。</p></li>     </ul>    </div>    <div class="tiledSection">     <h2 id="section597111654620">推送消息提示场景<i class="anchor-icon <h2 id="section2583253453">业务流程<i class="anchor-icon anchor-icon-link" anchorid="section2583253453" tips="复制节点链接"></i></h2>          <p><span><img height="502.74" originheight="720" originwidth="1280" src="https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20251211114456.51238060383575168462433053094849:50001231000000:2800:DFAD4DE511B1DE44874FFC803475FC5F4E29799F7358091A97BF9E6FF5EFDA61.png" title="点击放大" width="893.76"></span></p>     <p>使用Push Kit的主要业务流程如下：</p>     <ol>      <li>应用调用Push Kit，获取Push Token。</li>      <li>应用成功获取Token后，建议及时上报Token等信息至应用服务端。</li>      <li>应用服务端向华为Push Kit服务端（Push Cloud）发送推送消息请求。应用的通知开关默认关闭，发送请求前，请先请求通知授权，详情请参见<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/notification-enable" target="_blank">请求通知授权</a>。</li>      <li>Push Kit服务端下发消息到Push Kit。</li>      <li>Push Kit进行消息处理。</li>     </ol>    </div>    <div class="tiledSection">     <h2 id="section15315537123318">约束和限制<i class="anchor-icon anchor-icon-link" anchorid="section15315537123318" tips="复制节点链接"></i></h2>         </div>    <div class="tiledSection">     <h3 id="section8461125516338" class="firsth2">影响送达率的因素说明<i class="anchor-icon anchor-icon-link" anchorid="section8461125516338" tips="复制节点链接"></i></h3>          <p>Push Kit致力于提供安全可靠的系统级消息发送通道，保障消息成功送达。影响消息送达率的因素如下：</p>     <ul>      <li>终端设备是否在线。如果设备离线，Push Kit会缓存消息，待设备上线后，再将消息推送给设备。</li>      <li>终端设备上应用是否被卸载。</li>      <li>终端设备的网络状况是否稳定。</li>      <li>终端设备的安全控制策略。</li>     </ul>    </div>    <div class="tiledSection">     <h3 id="section1219154911359">推送消息的及时性<i class="anchor-icon anchor-icon-link" anchorid="section1219154911359" tips="复制节点链接"></i></h3>          <p>在终端设备网络条件良好且不拥堵情况下，Push Kit将使用智能推送策略以减少推送消息的时延。</p>     <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">       <p>为降低对用户的打扰，系统会学习用户的行为习惯，预测用户的睡眠时间，在用户睡眠期间实施消息管控。在此期间推送服务将暂时缓存该时间段内收到的消息（应用内通话或category=VoIP的消息除外）。用户结束睡眠后，推送服务会将消息重新投递到对应设备。</p>      </div></div></div>    </div>    <div class="tiledSection">     <h3 id="section0327131315367">推送消息长度与数量限制<i class="anchor-icon anchor-icon-link" anchorid="section0327131315367" tips="复制节点链接"></i></h3>          <ul>      <li>消息体最大不能超过4096Bytes（不包括Token）。</li>      <li>消息发送量，测试消息（参考消息体pushOptions.<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/push-scenariozed-api-request-param#section418321011212" target="_blank">testMessage</a>）每个项目限制所有应用共享1000条/天，正式消息区分场景有不同的配额，参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/push-msg-freq-control" target="_blank">消息频控</a>说明。</li>     </ul>    </div>    <div class="tiledSection">     <h3 id="section166483411411">网络受限说明<i class="anchor-icon anchor-icon-link" anchorid="section166483411411" tips="复制节点链接"></i></h3>          <p>如果终端设备连接的网络配置了防火墙或处于受限的网络下，将会影响消息的送达率，请检查以下端口号是否被禁用。</p>     <p>端口号：</p>     <ul>      <li>5223</li>      <li>443</li>     </ul>     <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">       <p>终端设备连接的推送服务器的IP是动态分配的，无法通过配置IP白名单方式放行。建议连接不受限的网络或放通以上端口。</p>      </div></div></div>    </div>    <div class="tiledSection">     <h3 id="section215725413308">支持的国家/地区<i class="anchor-icon anchor-icon-link" anchorid="section215725413308" tips="复制节点链接"></i></h3>          <p>Push Kit当前<a    <div class="tiledSection">     <h3 id="section1648815177260">模拟器版本和云真机说明<i class="anchor-icon anchor-icon-link" anchorid="section1648815177260" tips="复制节点链接"></i></h3>          <p>Push     <ul>      <li>Push Kit建立了从云端到终端的消息推送通道，支持开发者从云侧实时推送消息。如果开发者希望从本地推送通知，可通过<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/notification-overview" target="_blank">Notification href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/formkit-overview" target="_blank">Form Kit（卡片开发服务）</a>提前创建应用的服务卡片。</li>      <li>开发者<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/push-update-liveview">推送实况窗更新消息</a>时，需要通过<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/liveview-introduction" target="_blank">Live View Kit（实况窗服务）</a>提前创建本地实况窗。</li>      <li>开发者<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/push-voip">推送应用内通话消息</a>时，通过<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/call-introduction" target="_blank">Call Service Kit（通话服务）</a>管理应用通话能力。</li>     </ul>    </div>    <div class="tiledSection">     <h2 id="section1381417149521">示例代码<i class="anchor-icon anchor-icon-link"
