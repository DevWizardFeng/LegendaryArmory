<h1 _ngcontent-bcl-c119="" class="doc-title ng-star-inserted" title="Session间缓存共享"> Session间缓存共享 </h1>

<div _ngcontent-bcl-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>从6.0.0(20)开始，支持Session间缓存共享。</p>    <p>Session之间数据是隔离的，当业务场景需要实现跨Session访问缓存数据时，可通过以下两种标准方式达成目标：</p>    <ul>     <li>引用相同的HTTP响应缓存实例，不同Session能直接访问同一缓存池中的数据；</li>     <li>配置相同的缓存存储路径，使各个Session基于同一存储位置读写缓存，以此实现缓存数据的跨Session共享与交互。</li>
