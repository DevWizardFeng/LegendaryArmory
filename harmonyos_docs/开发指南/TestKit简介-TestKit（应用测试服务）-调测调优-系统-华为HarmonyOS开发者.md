<h1 _ngcontent-tjk-c119="" class="doc-title ng-star-inserted" title="Test Kit简介"> Test Kit简介 </h1>

<div _ngcontent-tjk-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>Test Kit为开发者提供了自动化测试框架，框架提供单元、UI和性能测试能力，支持编写ArkTS语言的单元、UI和性能自动化测试脚本，并通过测试结果查看相应功能的实现效果。从API version 20开始，支持性能测试能力。</p> <ul><li>单元测试能力：提供自动化测试的基础接口和运行机制，具体使用方法请参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/unittest-guidelines" target="_blank">单元测试框架使用指导</a>，主要能力如下：<ul><li>提供自动化用例定义接口，包括测试套定义、测试用例定义。</li><li>提供自动化用例断言接口，支持多种方式的断言能力，支持开发者在自动化脚本中灵活使用进行用例断言。</li><li>提供预置动作/清理动作执行接口，且支持测试套层级和测试用例层级的执行方式。</li><li>提供多种用例执行模式，包括筛选指定测试用例执行、随机执行、压力执行等。</li></ul>  </li><li>UI测试能力：提供UI自动化测试能力，测试脚本需基于单元测试框架开发，具体使用方法请参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/uitest-guidelines" target="_blank">UI测试框架使用指导</a>，主要能力如下：<ul><li>提供控件查找接口，支持多种控件查找方式，如按照控件属性查找、按照控件相对位置查找等。</li><li>提供模拟UI操作接口，支持多种类型如点击、双击、滑动、双指捏合等操作，也支持模拟多种外设如鼠标、键盘的操作。</li><li>提供模拟窗口操作接口，支持针对窗口进行模拟调整大小、移动等操作。</li><li>提供shell命令方式模拟UI操作能力，支持如点击、双击、滑动等操作。</li><li>提供监听系统弹框/toast能力，并可获取提示文本。</li></ul>  </li><li>性能测试能力：提供白盒性能自动化测试能力，测试脚本需基于单元测试框架开发，具体使用方法请参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/perftest-guideline" target="_blank">白盒性能测试框架使用指导</a>，主要能力如下：<ul><li>提供针对指定代码段执行过程的基础性能数据采集能力，包括执行耗时、执行CPU和内存占用数据等。</li><li>提供对应用场景化性能数据的采集，包括应用启动时延、页面切换时延、列表滑动帧率等。</li></ul>  </li></ul> </div> <div></div></div>
