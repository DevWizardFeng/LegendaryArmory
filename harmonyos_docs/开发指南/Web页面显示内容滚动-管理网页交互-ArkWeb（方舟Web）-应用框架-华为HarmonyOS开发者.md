<h1 _ngcontent-wvf-c119="" class="doc-title ng-star-inserted" title="Web页面显示内容滚动"> Web页面显示内容滚动 </h1>

<div _ngcontent-wvf-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>当Web页面的内容高度或宽度超过可视区域时，页面才能滚动。Web页面滚动有多种方式，包括使用外接设备、ArkTS侧接口调用和js侧接口调用。</p>    <div class="tiledSection">     <h2 id="使用外接设备控制web页面滚动">使用外接设备控制Web页面滚动<i class="anchor-icon anchor-icon-link" anchorid="使用外接设备控制web页面滚动" tips="复制节点链接"></i></h2>          <p>可以使用以下方式，通过触屏、触摸板和鼠标滚动控制Web页面滚动。</p>     <ul>      <li>通过触屏控制Web页面滚动：支持在触摸屏上单指上下左右滑动可以控制页面滚动。</li>      <li>通过触摸板控制Web页面滚动：支持在笔记本触摸板或者外接触摸板双指上下左右滑动，可以控制页面滚动。</li>      <li>通过鼠标滚轮控制Web页面滚动：支持用鼠标滚轮上下滑动来控制页面滚动</li>     </ul>    </div>    <div class="tiledSection">     <h2
