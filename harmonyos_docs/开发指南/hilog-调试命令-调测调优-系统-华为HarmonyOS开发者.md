<h1 _ngcontent-rqb-c119="" class="doc-title ng-star-inserted" title="hilog"> hilog </h1>

<div _ngcontent-rqb-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>HiLog日志系统，提供给系统框架、服务、以及应用，用于打印日志，记录用户操作、系统运行状态等。开发者可以通过hilog命令行查询相关日志信息。</p>    <div class="tiledSection">     <h2 id="环境要求">环境要求<i class="anchor-icon anchor-icon-link" anchorid="环境要求" tips="复制节点链接"></i></h2>          <ul>      <li>       <p>根据hdc命令行工具指导，完成<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/hdc#环境准备">环境准备</a>。</p></li>      <li>       <p>确保设备已正常连接，并执行hdc shell。</p></li>     </ul>    </div>    <div class="tiledSection">     <h2 id="hilog概述">hilog概述<i class="anchor-icon anchor-icon-link" anchorid="hilog概述" tips="复制节点链接"></i></h2>         </div>    <div class="tiledSection">     <h3 id="hilog日志生成" class="firsth2">hilog日志生成<i class="anchor-icon anchor-icon-link" anchorid="hilog日志生成" tips="复制节点链接"></i></h3>          <p>业务代码中，如果调用了HiLog接口，那么业务运行时就会生成对应的hilog日志。</p>    </div>    <div class="tiledSection">     <h3 id="hilog日志查看">hilog日志查看<i class="anchor-icon anchor-icon-link" anchorid="hilog日志查看" tips="复制节点链接"></i></h3>          <p><strong>在线查看</strong></p>     <ul>      <li>       <p>使用hdc shell hilog命令在线查看日志。</p></li>      <li>       <p>使用DevEco Studio工具在线查看日志，参考：<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-setup-hilog" target="_blank">日志分析</a>。</p></li>     </ul>     <p><strong>查看落盘的文件</strong></p>     <ul>      <li>       <p><a href="/consumer/cn/doc/harmonyos-guides/hilog#查看log版本信息">log版本</a>：hilog日志默认开启落盘，不同产品之间落盘数量可能存在差异，具体可以使用<a href="/consumer/cn/doc/harmonyos-guides/hilog#查看和设置落盘任务">查看和设置落盘任务</a>命令进行查询。</p></li>      <li>       <p><a href="/consumer/cn/doc/harmonyos-guides/hilog#查看log版本信息">nolog版本</a>：hilog日志默认不落盘，需要手动开启落盘，参考<a href="/consumer/cn/doc/harmonyos-guides/hilog#查看和设置落盘任务">查看和设置落盘任务</a>；设备打开开发者模式后自动开启落盘。</p></li>      <li>       <p>日志保存位置：设备data/log/hilog目录下，可以导出日志进行查看。导出及解析工具参考：<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/hilog-tool" target="_blank">hilogtool</a>。</p></li>     </ul>    </div>    <div class="tiledSection">     <h3
