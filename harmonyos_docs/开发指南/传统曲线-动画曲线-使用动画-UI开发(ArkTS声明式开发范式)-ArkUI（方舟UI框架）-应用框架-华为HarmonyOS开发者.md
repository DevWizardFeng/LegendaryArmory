<h1 _ngcontent-anc-c119="" class="doc-title ng-star-inserted" title="传统曲线"> 传统曲线 </h1>

<div _ngcontent-anc-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>传统曲线基于数学公式，创造形状符合开发者预期的动画曲线。以三阶贝塞尔曲线为代表，通过调整曲线控制点，可以改变曲线形状，从而带来缓入、缓出等动画效果。对于同一条传统曲线，由于不具备物理含义，其形状不会因为用户行为发生任何改变，缺少物理动画的自然感和生动感。建议优先采用物理曲线创建动画，将传统曲线作为辅助用于极少数必要场景中。</p>    <p>ArkUI提供了贝塞尔曲线、阶梯曲线等传统曲线接口，开发者可参照<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-curve" target="_blank">插值计算</a>进行查阅。</p>    <p>传统曲线的示例和效果如下：</p>    <div
