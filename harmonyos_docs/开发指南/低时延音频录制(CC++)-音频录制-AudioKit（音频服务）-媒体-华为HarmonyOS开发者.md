<h1 _ngcontent-oyq-c119="" class="doc-title ng-star-inserted" title="低时延音频录制(C/C++)"> 低时延音频录制(C/C++) </h1>

<div _ngcontent-oyq-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>从API version 10开始支持低时延音频录制。</p>    <p>低时延音频录制是一种通过软硬芯协同设计实现的音频渲染方案。其核心机制是通过减少buffer大小、优化读写数据架构，使该模式下音频录制具有更低的时延。</p>    <div class="tiledSection">     <h2 id="zh-cn_topic_0000002512608487_使用前提">使用前提<i class="anchor-icon anchor-icon-link" anchorid="zh-cn_topic_0000002512608487_使用前提" tips="复制节点链接"></i></h2>          <ul>      <li>支持低时延模式的音频输入设备。</li>      <li>可通过<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-audiocapturer-h#oh_audiocapturer_getfaststatus" target="_blank">OH_AudioCapturer_GetFastStatus()</a>验证当前设备是否支持低时延模式。</li>     </ul>    </div>    <div class="tiledSection">     <h2 id="zh-cn_topic_0000002512608487_开发指导">开发指导<i class="anchor-icon anchor-icon-link" anchorid="zh-cn_topic_0000002512608487_开发指导" tips="复制节点链接"></i></h2>         </div>    <div class="tiledSection">     <h3 id="zh-cn_topic_0000002512608487_简介" class="firsth2">简介<i class="anchor-icon anchor-icon-link" anchorid="zh-cn_topic_0000002512608487_简介" tips="复制节点链接"></i></h3>          <p>为使用低时延模式，开发者需要参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/using-ohaudio-for-recording">使用OHAudio开发音频录制功能(C/C++)</a>进行音频开发。</p>     <p>当前OHAudio支持两种模式：普通模式（AUDIOSTREAM_LATENCY_MODE_NORMAL）和低时延模式（AUDIOSTREAM_LATENCY_MODE_FAST）。</p>    </div>    <div class="tiledSection">     <h3 id="zh-cn_topic_0000002512608487_设置低时延模式">设置低时延模式<i target="_blank">OH_AudioStreamBuilder_SetLatencyMode()</a>，设置<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-audiostream-base-h#oh_audiostream_latencymode"     <h2 id="zh-cn_topic_0000002512608487_注意事项">注意事项<i class="anchor-icon anchor-icon-link" anchorid="zh-cn_topic_0000002512608487_注意事项" tips="复制节点链接"></i></h2>         </div>    <div class="tiledSection">     <h3 id="zh-cn_topic_0000002512608487_低时延模式限制" class="firsth2">低时延模式限制<i class="anchor-icon anchor-icon-link" anchorid="zh-cn_topic_0000002512608487_低时延模式限制" tips="复制节点链接"></i></h3>          <p>在以下场景中，即使设置了低时延模式，系统仍会使用普通模式。</p>     <ul>      <li>当前设备不支持低时延模式。</li>      <li>当前流格式不支持低时延模式。</li>      <li>系统低时延资源已被全部占用。</li>      <li>当前系统中存在更高优先级流（如：VOIP通话）。</li>     </ul>     <p>从API version 20开始，支持低时延相关查询接口。</p>     <ul>      <li>开发者通过调用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-audiocapturer-h#oh_audiocapturer_getfaststatus" target="_blank">OH_AudioCapturer_GetFastStatus()</a>来获取音频录制流是否正在低时延状态下工作。</li>      <li>       <ul>        <li>在部分特殊场景（如：存在更高优先级流、当前连接设备不支持等）下，开发者可以通过调用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-audiocapturer-h#oh_audiocapturer_onfaststatuschange" target="_blank">OH_AudioCapturer_OnFastStatusChange()</a>来获取低时延状态改变事件。</li>       </ul></li>     </ul>    </div>    <div class="tiledSection">     <h3 id="zh-cn_topic_0000002512608487_使用低时延流的场景">使用低时延流的场景<i class="anchor-icon anchor-icon-link" anchorid="zh-cn_topic_0000002512608487_使用低时延流的场景" tips="复制节点链接"></i></h3>          <ul>      <li>k歌、直播等对时延要求较高的场景，建议使用低时延模式。</li>      <li>普通录音、录像、录屏等没有实时要求的场景，不建议使用低时延模式。</li>     </ul>    </div>    <div class="tiledSection">     <h3 id="zh-cn_topic_0000002512608487_确保数据及时读入">确保数据及时读入<i class="anchor-icon anchor-icon-link" anchorid="zh-cn_topic_0000002512608487_确保数据及时读入" tips="复制节点链接"></i></h3>          <p>低时延模式下，应用读取数据的频次比普通录制模式高，如果获取数据不及时可能导致杂音等问题。开发者应避免在数据回调线程中做耗时操作，确保数据回调线程可以及时返回。</p>    </div>    <div class="tiledSection">     <h3 id="zh-cn_topic_0000002512608487_数据回调线程">数据回调线程<i class="anchor-icon anchor-icon-link" anchorid="zh-cn_topic_0000002512608487_数据回调线程" tips="复制节点链接"></i></h3>          <p>音频录制的音频数据需要通过回调接口读入。开发者要实现回调接口，使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-audiostreambuilder-h#oh_audiostreambuilder_setcapturerreaddatacallback" target="_blank">OH_AudioStreamBuilder_SetCapturerReadDataCallback</a>设置回调函数，在设置音频回调函数时，回调函数<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-audiocapturer-h#oh_audiocapturer_onreaddatacallback" target="_blank">OH_AudioCapturer_OnReadDataCallback</a>（从API version 12开始支持）用于读取音频数据。</p>     <p>开发音频录制功能的示例代码请参考：<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/using-ohaudio-for-recording">使用OHAudio开发音频录制功能(C/C++)</a>。</p>
