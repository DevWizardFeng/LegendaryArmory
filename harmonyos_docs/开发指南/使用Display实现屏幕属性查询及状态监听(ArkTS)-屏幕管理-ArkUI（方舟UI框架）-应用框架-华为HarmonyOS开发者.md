<h1 _ngcontent-sbv-c119="" class="doc-title ng-star-inserted" title="使用Display实现屏幕属性查询及状态监听 (ArkTS)"> 使用Display实现屏幕属性查询及状态监听 (ArkTS) </h1>

<div _ngcontent-sbv-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <div class="tiledSection">     <h2 id="场景介绍">场景介绍<i class="anchor-icon anchor-icon-link" anchorid="场景介绍" tips="复制节点链接"></i></h2>          <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-display" target="_blank">Display</a>屏幕属性提供管理设备屏幕的一些基础能力，例如获取默认显示设备的相关信息、获取全部显示设备的信息，此外还能对显示设备的插拔行为进行监听。应用可以根据对应的屏幕信息、屏幕状态变化、屏幕折叠状态等适配不同的UI界面显示。</p>     <p>屏幕属性的常见使用场景有以下几种：</p>     <ul>      <li>查询屏幕信息：包括屏幕的分辨率、物理像素密度、逻辑像素密度、刷新率、屏幕尺寸、屏幕旋转方向、屏幕旋转角度等，具体可见<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-display#属性" target="_blank">Display属性</a>。</li>      <li>监听屏幕状态变化，包括屏幕旋转变化，屏幕分辨率变化、屏幕刷新率变化等。</li>      <li>查询当前设备是否为可折叠设备，同时支持折叠状态（展开/折叠）变化的监听。</li>     </ul>    </div>    <div class="tiledSection">     <h2 id="接口说明">接口说明<i class="anchor-icon
