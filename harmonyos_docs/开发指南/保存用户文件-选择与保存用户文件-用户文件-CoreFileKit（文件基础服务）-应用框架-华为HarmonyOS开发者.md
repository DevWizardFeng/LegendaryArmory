<h1 _ngcontent-ecq-c119="" class="doc-title ng-star-inserted" title="保存用户文件"> 保存用户文件 </h1>

<div _ngcontent-ecq-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>在从网络下载文件到本地或将已有用户文件另存为新的文件路径等场景下，需要使用FilePicker提供的保存用户文件的能力。需关注以下关键点：</p>    <p><strong>权限说明</strong></p>    <ul>     <li>通过Picker获取的URI默认只具备<strong>临时读写权限</strong>，临时授权在应用退出后台自动失效。</li>     <li>获取持久化权限需要通过<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/file-persistpermission#通过picker获取临时授权并进行授权持久化">FilePicker设置永久授权</a>方式获取。</li>     <li>使用Picker对音频、图片、视频、文档类文件的保存操作<strong>无需申请权限</strong>。</li>    </ul>    <p><strong>系统隔离说明</strong></p>    <ul>     <li>通过Picker保存的文件存储在用户指定的目录。此类文件与图库管理的资源隔离，无法在图库中看到。</li>     <li>若开发者需要保存图片、视频资源到图库，可使用用户无感的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/photoaccesshelper-savebutton#使用安全控件保存媒体库资源">安全控件进行保存</a>。</li>    </ul>    <div class="tiledSection">     <h2 id="保存图片或视频类文件">保存图片或视频类文件<i class="anchor-icon anchor-icon-link" anchorid="保存图片或视频类文件" tips="复制节点链接"></i></h2>          <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/photoaccesshelper-savebutton">Media Library Kit（媒体文件管理服务）中能力来保存媒体库资源</a>。</p>     <p>如果开发场景无法调用安全控件进行图片、视频保存，可使用相册管理模块<a
