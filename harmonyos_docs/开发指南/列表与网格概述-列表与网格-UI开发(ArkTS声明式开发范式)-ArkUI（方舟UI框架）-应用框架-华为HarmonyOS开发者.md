<h1 _ngcontent-ild-c119="" class="doc-title ng-star-inserted" title="列表与网格概述"> 列表与网格概述 </h1>

<div _ngcontent-ild-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>许多应用存在滚动展示同类项目集合的需求，例如显示图片、视频、音乐、新闻、商品等。此类场景可以根据项目排列方式分别选择<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-layout-development-create-list">List</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-layout-development-create-grid">Grid</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-layout-development-create-waterflow">WaterFlow</a>实现，在圆形屏幕推荐使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-layout-development-create-arclist">ArcList</a>。</p>    <div class="tiledSection">     <h2 id="列表">列表<i class="anchor-icon anchor-icon-link" anchorid="列表" tips="复制节点链接"></i></h2>          <p>List适合单列和多列宽度相同的场景，如通讯录、音乐列表、购物清单等。</p>     <p>直播评论、即时聊天等应用场景需要在列表底部插入数据时，内容应自动向上滚动，以展示新插入的节点，此功能可通过配置<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-list#stackfromend19" target="_blank">List从尾部开始布局</a>实现。</p>     <p><span><img originheight="334" originwidth="1036" src="https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20251211114751.12443153923951146591498150930279:50001231000000:2800:812BF35EF56F1EBD9F6F01C112C9E3F2A3BE00DAC2746E4BA7D54B96D9B10D60.png" width="920" height="296.6023166023166"></span></p>    </div>    <div class="tiledSection">     <h2 id="网格">网格<i class="anchor-icon anchor-icon-link" anchorid="网格" tips="复制节点链接"></i></h2>          <p>网格布局由“行”和“列”分割的单元格组成，通过指定“项目”所在单元格实现多种布局，应用场景包括九宫格图片展示、日历、计算器等。</p>     <p>对于部分项目占用多行或多列的场景，可以通过在创建Grid时传入合适的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-grid#gridlayoutoptions10对象说明" target="_blank">GridLayoutOptions</a>来实现。</p>     <p><span><img originheight="465" originwidth="2025" src="https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20251211114751.40789679163338303303501208732009:50001231000000:2800:B8E488AC9928F02EDC99C5DE16489DF2829A0F7AE0A606D827C364299D7B57E0.png" width="920" height="211.25925925925927"></span></p>    </div>    <div class="tiledSection">     <h2 id="瀑布流">瀑布流<i class="anchor-icon anchor-icon-link" anchorid="瀑布流" tips="复制节点链接"></i></h2>          <p>瀑布流布局是一种多列等宽但高度不等的布局方式，适用于需要错落排列的场景，如图片和视频展示、商品推荐等。</p>     <p>同一个页面内有不同列数分段混合布局的场景，可以通过设置<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-waterflow#waterflowoptions对象说明" target="_blank">WaterFlowSections</a>实现。</p>     <p><span><img originheight="779" originwidth="377" src="https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20251211114751.67545664657311393374390846134372:50001231000000:2800:F35640E1A4FCE7DA25AEA40031351EEE793BFA969CE1A34583F4CBDE4B1A2693.png" width="377" height="779"></span></p>    </div>    <div class="tiledSection">     <h2 id="弧形列表">弧形列表<i class="anchor-icon anchor-icon-link" anchorid="弧形列表" tips="复制节点链接"></i></h2>          <p>弧形列表是一种专为圆形屏幕设备设计的特殊列表，支持列表项在接近屏幕上下两端自动缩放的效果。</p>     <p><span><img originheight="270" originwidth="266"
