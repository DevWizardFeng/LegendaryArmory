<h1 _ngcontent-fai-c119="" class="doc-title ng-star-inserted" title="坚盾守护模式"> 坚盾守护模式 </h1>

<div _ngcontent-fai-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>坚盾守护模式提供给高安全需求用户的系统级别安全模式。该模式通过限制设备基础功能，增强安全性，有效抵御远程攻击面的针对性攻击。</p>  <div class="tiledSection"><h2 id="arkweb限制的html5特性">ArkWeb限制的HTML5特性<i class="anchor-icon anchor-icon-link" anchorid="arkweb限制的html5特性" tips="复制节点链接"></i></h2><p>坚盾守护模式开启时，ArkWeb通过限制以下HTML5特性减少攻击面。</p> <ul><li><p>禁止使用WebAssembly能力。</p>  </li><li><p>禁止使用WebGL、WebGL2能力。</p>  </li><li><p>禁止使用PDF Viewer预览功能。</p>  </li><li><p>禁止使用MathML能力。</p>  </li><li><p>禁止使用Web Speech API语音识别能力。</p>  </li><li><p>禁止使用RTCDataChannel接口。</p>  </li><li><p>禁止使用MediaDevices.getUserMedia接口提示用户允许访问媒体输入设备。</p>  </li><li><p>禁止使用Service Worker能力。</p>  </li><li><p>禁止使用非代理UDP流量，防止WebRTC泄露真实源ip。</p>  </li><li><p>禁止即时编译（JIT）能力。</p>  </li></ul> </div>  <div class="tiledSection"><h2 id="评估对应用的影响">评估对应用的影响<i class="anchor-icon anchor-icon-link" anchorid="评估对应用的影响" tips="复制节点链接"></i></h2><p>要评估应用在坚盾守护模式下的受影响程度及兼容性，可前往“设置 &gt; 隐私和安全 &gt; 坚盾守护模式”开启。</p> <p><span><img originheight="931" originwidth="470" src="https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20251211114739.70592192483029755939628118398506:50001231000000:2800:0C91C5128F56E3AB88B3768AE83E70F332BF2BC9E7BA707F2BA54BA43AA20DD4.png" width="470" height="931"></span></p> <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content"> <ul><li>如果需要评估调试版本（未上架应用市场）应用的兼容性，先开启开发者选项，然后开启坚盾守护模式。</li><li>可以采用Device Security Kit提供的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/devicesecurity-securitymode" target="_blank">查询设备安全模式(C/C++)能力</a>，判断坚盾守护模式是否开启。</li></ul>  </div></div></div> <p>运行应用的相应功能后，可通过以下方式确认是否受影响：</p> <ul><li><p>排查前端代码是否存在WebAssembly相关接口调用，WebAssembly用于在Web上运行C/C++等低级语言编译代码，坚盾守护模式下无法调用。</p>  </li><li><p>排查前端代码是否存在WebGL相关接口调用，WebGL提供3D图形绘制能力。在坚盾守护模式下，相关接口无法调用。</p>  </li><li><p>排查是否有在线显示PDF的功能场景，坚盾守护模式下无法在线显示PDF，例如通过loadUrl加载PDF链接。</p>  </li><li><p>排查HTML页面是否存在&lt;math&gt;标签嵌入的MathML语法，坚盾守护模式下，MathML语法无法解析，导致显示异常。</p>  </li><li><p>排查前端代码是否存在SpeechRecognition（语音识别）、SpeechSynthesis（语音合成）等接口调用，坚盾守护模式下，相关接口无法调用。</p>  </li><li><p>排查前端代码是否存在RTCDataChannel/createDataChannel等接口调用，该类接口属于WebRTC API，用于建立双向数据通道，实现对等端之间的实时数据交换。坚盾守护模式下，相关接口无法调用。</p>  </li><li><p>排查前端代码是否存在MediaDevices.getUserMedia接口调用，该接口用于请求访问流媒体设备，如摄像头和麦克风。在坚盾守护模式下，相关接口调用会抛出异常：“can't use getUserMedia on advancedSecurityMode!”</p>  </li><li><p>排查前端代码是否存在ServiceWorker相关接口调用，该机制实现离线缓存、网络请求拦截和推送通知等功能，在坚盾守护模式下无法创建。</p>  </li><li><p>在坚盾守护模式下，WebRTC禁止使用非代理UDP传输。应用需验证WebRTC场景下的网络功能和性能。</p>  </li><li><p>坚盾守护模式下应用需评估JS性能。</p>  </li></ul> </div> </div> <div></div></div>
