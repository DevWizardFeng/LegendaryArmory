    <p>装饰器： 用于装饰类、结构、方法以及变量，并赋予其特殊的含义。如上述示例中@Entry、@Component和@State都是装饰器，<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-create-custom-components#entry">@Entry</a>表示该自定义组件为入口组件，<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-state">@State</a>表示组件中的状态变量，状态变量变化会触发UI刷新。</p></li>     <li>      <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-declarative-ui-description">UI描述</a>：以声明式的方式来描述UI的结构，例如build()方法中的代码块。</p></li>     <li>      <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-create-custom-components">自定义组件</a>：可复用的UI单元，可组合其他组件，如上述被@Component装饰的struct Hello。</p></li>     <li>      <p>系统组件：ArkUI框架中默认内置的基础和容器组件，可以直接调用，例如示例中的Column、Text、Divider、Button。</p></li>     <li>      <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-component-general-attributes" target="_blank">属性方法</a>：组件可以通过链式调用配置多项属性，如fontSize()、width()、height()、backgroundColor()等。</p></li>     <li>      <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-component-general-events" target="_blank">事件方法</a>：组件可以通过链式调用设置多个事件的响应逻辑，如跟随在Button后面的onClick()。</p></li>    </ul>    <p>除此之外，ArkTS扩展了多种语法范式来使开发更加便捷：</p>    <ul>     <li>      <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-builder">@Builder</a>/<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-builderparam">@BuilderParam</a>：特殊的封装UI描述的方法，细粒度的封装和复用UI描述。</p></li>     <li>      <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-extend">@Extend</a>/<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-style">@Styles</a>：扩展系统组件和封装属性样式，更灵活地组合系统组件。</p></li>     <li>      <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-statestyles">stateStyles</a>：多态样式，可以依据组件的内部状态的不同，设置不同样式。</p></li>    </ul>   </div>   <div></div></div>
