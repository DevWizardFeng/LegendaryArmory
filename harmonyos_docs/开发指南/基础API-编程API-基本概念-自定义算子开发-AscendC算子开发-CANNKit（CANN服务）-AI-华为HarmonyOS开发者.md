<div _ngcontent-rhm-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><div class="tiledSection"><h2 id="section201612564012">数据搬运<i class="anchor-icon anchor-icon-link" anchorid="section201612564012" tips="复制节点链接"></i></h2><p>数据搬运接口，包括<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-common-data-movement">普通数据搬运</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-channel-associated-format-conversion">随路格式转换</a>。</p> <ul><li>普通数据搬运接口，适用于连续和不连续数据搬运。</li><li>随路格式转换接口，适用于在搬运时进行格式转换。</li></ul> </div> <div class="tiledSection"><h2 id="section1876762615403">内存管理与同步控制<i class="anchor-icon anchor-icon-link" anchorid="section1876762615403" tips="复制节点链接"></i></h2><p>AscendC编程范式，把算子核内的处理程序，分成多个流水任务，通过队列(Queue)完成<strong>任务间通信和同步</strong>，并通过统一的<strong>资源管理</strong>模块(Pipe)来统一管理内存、事件等资源。</p> <p>AscendC提供一组内存管理与同步控制API，开发者使用这一组API即可完成任务间同步和内存管理。</p> <p>核心的API包括：</p> <ul><li>AllocTensor：从Queue中分配Tensor，Tensor所占大小为InitBuffer时设置的每块内存长度。</li><li>FreeTensor：释放Queue中的指定Tensor，供Queue后续使用。</li><li>EnQue：将Tensor push到队列Queue。</li><li>DeQue：将Tensor从队列Queue中取出，用于后续处理。</li></ul> </div> </div> <div></div></div>
