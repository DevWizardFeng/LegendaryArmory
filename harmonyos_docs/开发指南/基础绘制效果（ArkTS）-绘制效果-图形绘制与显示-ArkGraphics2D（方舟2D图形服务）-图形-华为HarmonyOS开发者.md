<div _ngcontent-jnh-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <div class="tiledSection">     <h2 id="场景介绍">场景介绍<i class="anchor-icon anchor-icon-link" anchorid="场景介绍" tips="复制节点链接"></i></h2>          <p>在进行绘制时，可以进行一些基础效果的设置，比如设置填充颜色、设置抗锯齿、设置图形描边、设置图形线条连接样式等。</p>     <p>主要通过画刷（Brush）设置填充基础效果，通过画笔（Pen）设置描边基础效果。</p>    </div>    <div class="tiledSection">     <h2 id="填充效果">填充效果<i class="anchor-icon anchor-icon-link" anchorid="填充效果" tips="复制节点链接"></i></h2>
