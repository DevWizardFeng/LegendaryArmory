<div _ngcontent-mql-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <div class="tiledSection">     <h2 id="概述">概述<i class="anchor-icon anchor-icon-link" anchorid="概述" tips="复制节点链接"></i></h2>          <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-mediaquery" target="_blank">媒体查询</a>作为响应式设计的核心，在移动设备上应用十分广泛。媒体查询可根据不同设备类型或同设备不同状态修改应用的样式。媒体查询常用于下面两种场景：</p>     <ol>      <li>       <p>针对设备和应用的属性信息（比如显示区域、深浅色、分辨率），设计出相匹配的布局。</p></li>      <li>       <p>当屏幕发生动态改变时（比如分屏、横竖屏切换），同步更新应用的页面布局。</p></li>     </ol>    </div>    <div class="tiledSection">     <h2 id="引入与使用流程">引入与使用流程<i class="anchor-icon anchor-icon-link" anchorid="引入与使用流程" tips="复制节点链接"></i></h2>          <p>媒体查询通过mediaquery模块接口，设置查询条件并绑定回调函数，任一<a
