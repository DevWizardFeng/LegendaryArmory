<h1 _ngcontent-typ-c119="" class="doc-title ng-star-inserted" title="弧形列表 (ArcList)（圆形屏幕推荐使用）"> 弧形列表 (ArcList)（圆形屏幕推荐使用） </h1>

<div _ngcontent-typ-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>从API version 18开始支持弧形列表。弧形列表是一种专为圆形屏幕设备设计的特殊列表，它能够以结构化、可滚动的形式高效展示信息。具体用法可参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-arclist" target="_blank">ArcList</a>。</p>    <p>使用弧形列表可以通过在<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-arclist" target="_blank">ArcList</a>组件中按垂直方向线性排列子组件<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-arclistitem" target="_blank">ArcListItem</a>，可以为弧形列表中的每一项提供独立视图。此外，可以使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-rendering-control-foreach">循环渲染</a>来迭代一组列表项，或结合任意数量的单个视图与<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-rendering-control-foreach">ForEach</a>结构，构建复杂的弧形列表。<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-arclist" target="_blank">ArcList</a>组件支持多种<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-rendering-control-overview">渲染控制</a>方式，包括条件渲染、循环渲染和懒加载，以生成子组件。</p>    <div class="tiledSection">     <h2 id="创建弧形列表">创建弧形列表<i
