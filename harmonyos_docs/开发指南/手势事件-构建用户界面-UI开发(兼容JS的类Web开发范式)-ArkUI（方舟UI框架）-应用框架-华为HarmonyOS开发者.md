<h1 _ngcontent-lbt-c119="" class="doc-title ng-star-inserted" title="手势事件"> 手势事件 </h1>

<div _ngcontent-lbt-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>手势表示由单个或多个事件识别的语义动作（例如：触摸、点击和长按）。一个完整的手势也可能由多个事件组成，对应手势的生命周期。支持的事件有：</p> <p><strong>触摸</strong></p> <ul><li><p>touchstart：手指触摸动作开始。</p>  </li><li><p>touchmove：手指触摸后移动。</p>  </li><li><p>touchcancel：手指触摸动作被打断，如来电提醒、弹窗。</p>  </li><li><p>touchend：手指触摸动作结束。</p>  </li></ul> <p><strong>点击</strong></p> <p>click：用户快速轻敲屏幕。</p> <p><strong>长按</strong></p>
