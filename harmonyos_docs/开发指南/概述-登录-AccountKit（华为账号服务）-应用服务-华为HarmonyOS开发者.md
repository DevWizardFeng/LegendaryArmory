<div _ngcontent-vex-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <p>Account Kit提供了<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/account-phone-unionid-login">华为账号一键登录</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/account-unionid-login">华为账号登录</a>、<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/account-silent-login">静默登录</a>等多种登录方式，其中华为账号一键登录<strong>仅支持企业开发者</strong>使用，华为账号登录和静默登录<strong>既支持企业开发者也支持个人开发者</strong>使用，应用可根据实际场景选择使用其中一种或多种方式进行账号登录。</p>    <div class="tiledSection">     <h2 id="section1463117011369">基础概念<i class="anchor-icon anchor-icon-link" anchorid="section1463117011369" tips="复制节点链接"></i></h2>      </tbody></table></div>     </div>    </div>    <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">      <p>在开发HarmonyOS应用时，您需要考虑同一用户在非HarmonyOS应用和HarmonyOS应用的用户数据是否互通。如果您之前使用OpenID来关联用户数据，我们建议将用户数据关系切换成UnionID，以确保用户使用HarmonyOS应用后可以继承老版本的用户数据。具体切换指导可以参考：<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/account-api-get-unionid" target="_blank">通过OpenID获取UnionID</a>。</p>     </div></div></div>    <div class="tiledSection">     <h2 id="section615311504544">场景介绍<i class="anchor-icon anchor-icon-link" anchorid="section615311504544" tips="复制节点链接"></i></h2>          <ul>      <li><strong>获取手机号和UnionID登录，即华为账号一键登录</strong></li>     </ul>     <p>若应用需要同时获取手机号和UnionID，推荐使用此场景，用户仅需一次点击操作，应用即可获取用户<strong>手机号和</strong><strong>UnionID。</strong>应用获取到用户手机号和UnionID后，<strong>可同时通过手机号和UnionID与应用原有用户体系进行关联</strong>。本场景仅支持企业开发者使用，个人开发者请使用华为账号登录或静默登录方式。</p>     <ul>      <li><strong>华为账号登录</strong></li>     </ul>     <p>若应用只需要获取UnionID可以使用此场景<strong>。</strong>应用获取到用户UnionID后，<strong>可通过UnionID与应用原有用户体系进行关联</strong>。</p>     <ul>      <li><strong>静默登录</strong></li>     </ul>     <p>在应用卸载重装、用户换机等场景，应用可通过Account Kit提供的静默登录方式即不需要用户点击登录/注册按钮，即可获取用户的身份标识UnionID，完成用户的静默登录。</p>     <ul>      <li><strong>订阅华为账号登录/登出事件</strong></li>     </ul>     <p>当应用需要跟随华为账号的登录状态进行登录登出时，可以通过订阅华为账号的登录登出事件进行判断。</p>    </div>   </div>   <div></div></div>
