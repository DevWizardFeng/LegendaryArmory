<div _ngcontent-ydk-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <div class="tiledSection">     <h2 id="section148314404566">场景介绍<i class="anchor-icon anchor-icon-link" anchorid="section148314404566" tips="复制节点链接"></i></h2>          <p>从6.0.0(20) 版本开始，新增支持模拟点击检测。</p>     <p>应用通过调用Device Security Kit的detectSimulatedClickRisk接口，获取模拟点击检测结果，用于自动化点击、设备墙等作弊行为检测。</p>     <p>应用可以根据检测结果评估如何进行业务操作。</p>    </div>    <div class="tiledSection">     <h2 id="section105801826165716">约束与限制<i class="anchor-icon anchor-icon-link" anchorid="section105801826165716" tips="复制节点链接"></i></h2>          <p>每30秒最多可以调用10次，每个应用在每个设备上每天最多可以调用20次。</p>    </div>    <div class="tiledSection">     <h2 id="section15741164765720">业务流程<i class="anchor-icon anchor-icon-link" anchorid="section15741164765720" tips="复制节点链接"></i></h2>          <p><span><img originheight="575" originwidth="1124" width="920" height="470.6405693950178"></span></p>    </div>    <p><strong>流程说明：</strong></p>    <ol>     <li><strong>开发者应用调用detectSimulatedClickRisk接口，发起模拟点击检测请求。</strong>      <p>Device Security Kit收到请求后，首先采集当前设备模拟点击线索数据，然后将线索数据发送到Device Security服务器做检测，最后通过detectSimulatedClickRisk接口的返回值将检测结果传递给开发者应用。</p></li>     <li><strong>获取检测结果，并根据结果做出相应处理。</strong></li>    </ol>    <div
