<h1 _ngcontent-aoc-c119="" class="doc-title ng-star-inserted" title="管理应用窗口（Stage模型）"> 管理应用窗口（Stage模型） </h1>

<div _ngcontent-aoc-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <div class="tiledSection">     <h2 id="基本概念">基本概念<i class="anchor-icon anchor-icon-link" anchorid="基本概念" tips="复制节点链接"></i></h2>          <ul>      <li>       <p>窗口沉浸式能力：指对状态栏、导航栏等系统窗口进行控制，减少状态栏导航栏等系统界面的突兀感，从而使用户获得最佳体验的能力。沉浸式能力只在应用主窗口作为全屏窗口时生效。通常情况下，应用的辅助窗口（子窗、全局悬浮窗等）和处于自由窗口下的应用主窗口无法使用沉浸式能力。</p></li>      <li>       <p>全局悬浮窗：全局悬浮窗是一种特殊的应用辅助窗口，具备在应用主窗口和对应Ability退至后台后仍然可以在前台显示的能力。全局悬浮窗可以用于应用退至后台后，使用小窗继续显示UI，例如音乐应用用于显示桌面歌词等。应用在创建全局悬浮窗前，需要申请对应的权限。</p></li>     </ul>    </div>    <div class="tiledSection">     <h2 id="场景介绍">场景介绍<i class="anchor-icon anchor-icon-link" anchorid="场景介绍" tips="复制节点链接"></i></h2>          <p>在Stage模型下，管理应用窗口的典型场景有：</p>     <ul>      <li>       <p>设置应用主窗口属性及目标页面</p></li>      <li>       <p>设置应用子窗口属性及目标页面</p></li>      <li>       <p>体验窗口沉浸式能力</p></li>      <li>       <p>设置全局悬浮窗</p></li>      <li>       <p>监听窗口不可交互与可交互事件</p></li>     </ul>     <p>以下分别介绍具体开发方式。</p>    </div>    <div tips="复制节点链接"></i></h2>          <p>开发者可以按需创建应用子窗口，如弹窗等，并对其进行属性设置等操作。</p>     <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">       <p>以下几种场景不建议使用子窗口，建议优先考虑使用控件<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-overlay" target="_blank">overlay</a>能力实现。</p>       <ul>        <li>移动设备（手机、在非自由模式下的平板设备）场景下子窗不能超出处于悬浮窗模式、分屏模式的应用主窗口范围，与控件一致。</li>        <li>分屏窗口与自由窗口模式下，主窗口位置大小发生改变时控件实时跟随变化能力优于子窗。</li>        <li>部分设备平台下根据实际的系统配置限制，子窗只有系统默认的动效和圆角阴影，应用无法设置，自由度低。</li>       </ul>      </div></div></div>    </div>    <div class="tiledSection">     <h3 id="开发步骤-1" class="firsth2">开发步骤<i class="anchor-icon anchor-icon-link" anchorid="开发步骤-1" tips="复制节点链接"></i></h3>          <ol> <p>子窗口创建成功后，可以改变其大小、位置等，还可以根据应用需要设置窗口背景色、亮度等属性。</p>       <p>在调用showWindow之前，建议设置子窗口的大小和位置。</p>       <p>如果没有设置子窗口的大小，调用showWindow后:</p>       <ul>        <li><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/window-terminology#自由窗口">自由窗口</a>状态下，默认子窗口大小为当前物理屏幕的大小。其中，<a   </ul></li>      <li>       <p>加载显示子窗口的具体内容。</p>       <p>通过setUIContent和showWindow接口加载显示子窗口的具体内容。</p></li>      <li>       <p>销毁子窗口。</p>
