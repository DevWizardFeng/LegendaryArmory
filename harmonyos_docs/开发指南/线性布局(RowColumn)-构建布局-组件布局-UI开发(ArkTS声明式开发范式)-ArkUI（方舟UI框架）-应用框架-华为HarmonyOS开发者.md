<h1 _ngcontent-sbm-c119="" class="doc-title ng-star-inserted" title="线性布局 (Row/Column)"> 线性布局 (Row/Column) </h1>

<div _ngcontent-sbm-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <div class="tiledSection">     <h2 id="概述">概述<i class="anchor-icon anchor-icon-link" anchorid="概述" tips="复制节点链接"></i></h2>          <p>线性布局（LinearLayout）是开发中最常用的布局，通过线性容器<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-row" target="_blank">Row</a>和<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-column" target="_blank">Column</a>构建。线性布局是其他布局的基础，其子元素在线性方向上（水平方向和垂直方向）依次排列。线性布局的排列方向由所选容器组件决定，Row容器内子元素按照水平方向排列，Column容器内子元素按照垂直方向排列。根据不同的排列方向，开发者可选择使用Row或Column容器创建线性布局。</p>     <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">       <p>在复杂界面中使用多组件嵌套时，若布局组件的嵌套层数过深或嵌套的组件数量过多，将会产生额外开销。建议通过移除冗余节点、利用布局边界减少布局计算、合理采用渲染控制语法及布局组件方法来优化性能。最佳实践请参考<a href="https://developer.huawei.com/consumer/cn/doc/best-practices/bpta-improve-layout-performance" target="_blank">合理使用布局</a>。</p>      </div></div></div>     <p><strong>图1</strong> <li>       <p>布局容器：具有布局能力的容器组件，可以承载其他元素作为其子元素，布局容器会对其子元素进行尺寸计算和布局排列。</p></li>      <li>       <p>布局子元素：布局容器内部的元素。</p></li>      <li>       <p>主轴：线性布局容器在布局方向上的轴线，子元素默认沿主轴排列。Row容器主轴为水平方向，Column容器主轴为垂直方向（图示可参考弹性布局<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-layout-development-flex-layout#基本概念">基本概念</a>中的主轴）。</p></li>      <li>       <p>交叉轴：垂直于主轴方向的轴线。Row容器交叉轴为垂直方向，Column容器交叉轴为水平方向（图示可参考弹性布局<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-layout-development-flex-layout#基本概念">基本概念</a>中的交叉轴）。</p></li>      <li>       <p>间距：布局子元素的间距。</p></li>     </ul>    </div>    <div class="tiledSection">     <h2 id="布局子元素在排列方向上的间距">布局子元素在排列方向上的间距<i class="anchor-icon anchor-icon-link" anchorid="布局子元素在排列方向上的间距"      <p>自适应延伸是指在不同尺寸设备下，当页面的内容超出屏幕大小而无法完全显示时，可以通过滚动条进行拖动展示。对于线性布局，这种方法适用于容器中内容无法一屏展示的场景。通常有以下两种实现方式。</p>     <ul>      <li>       <p><a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-layout-development-create-list#添加滚动条">在List中添加滚动条</a>：当List子项过多一屏放不下时，可以将每一项子元素放置在不同的组件中，通过滚动条进行拖动展示。可以通过<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-scroll#scrollbar" target="_blank">scrollBar</a>属性设置滚动条的常驻状态，<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-scroll#edgeeffect" target="_blank">edgeEffect</a>属性设置拖动到内容最末端的回弹效果。</p></li>      <li>       <p>使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-container-scroll"
