id="section175232481311">场景介绍<i class="anchor-icon anchor-icon-link" anchorid="section175232481311" tips="复制节点链接"></i></h2>          <p>从5.1.0(18)版本开始， 导航组件新增支持标题栏<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ui-design-hdsnavigation#section21731754221" target="_blank">过渡模糊</a>与<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ui-design-hdsnavigation#section21731754221" target="_blank">渐变模糊</a>样式。</p>     <p>当应用开发者需要使用标题栏样式随内容区滚动而动态改变样式的导航组件时，可以通过设置titleBar属性中的<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ui-design-hdsnavigation#section1867665412513" target="_blank">style</a>配置，自定义滚动前后的标题栏样式。</p>    </div>    <div class="tiledSection">     <h3 id="section14322376130" class="firsth2">通用模糊样式<i class="anchor-icon anchor-icon-link"
