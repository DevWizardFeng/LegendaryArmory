<h1 _ngcontent-nyb-c119="" class="doc-title ng-star-inserted" title="设置自定义节点跨语言属性"> 设置自定义节点跨语言属性 </h1>

<div _ngcontent-nyb-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;">   <div>    <div class="tiledSection">     <h2 id="概述">概述<i class="anchor-icon anchor-icon-link" anchorid="概述" tips="复制节点链接"></i></h2>          <p>ArkUI支持在前端使用ArkTS语言创建命令式节点，即<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-framenode" target="_blank">FrameNode</a>节点，也可以在Native侧使用C语言创建命令式节点，并且可以混合使用两类节点构建页面。</p>     <p>针对上述场景，ArkUI提供命令式节点跨语言属性设置功能，即使用ArkTS语言创建的命令式节点，可以在Native侧进行属性设置。使用C语言创建的节点，可以在ArkTS侧进行属性设置。</p>     <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content">       <p>下述示例中，需要先进行Native侧配置，请参考<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ndk-access-the-arkts-page">接入ArkTS页面</a>完成。</p>      </div></div></div>    </div>    <div class="tiledSection">     <h2 id="设置和获取跨语言配置">设置和获取跨语言配置<i class="anchor-icon anchor-icon-link" anchorid="设置和获取跨语言配置" tips="复制节点链接"></i></h2>          <p>跨语言指的是跨越ArkTS语言和C语言。跨语言配置指的是命令式节点上对于跨语言操作的权限配置。</p>     <p>可以通过<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-framenode#setcrosslanguageoptions15" target="_blank">setCrossLanguageOptions</a>与<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-node-h#oh_arkui_nodeutils_setcrosslanguageoption" target="_blank">OH_ArkUI_NodeUtils_SetCrossLanguageOption</a>接口设置当前节点的跨语言配置。如果当前节点无法修改或设置跨语言配置，则会抛出异常信息。</p>     <p>可以使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-arkui-framenode#getcrosslanguageoptions15" target="_blank">getCrossLanguageOptions</a>与<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-node-h#oh_arkui_nodeutils_getcrosslanguageoption" target="_blank">OH_ArkUI_NodeUtils_GetCrossLanguageOption</a>接口获取当前节点的跨语言配置。</p>     <p>以下示例描述了如何设置和获取ArkTS命令式节点的跨语言配置。</p>     <div _ngcontent-nyb-c106="" class="highlight-div"><div
