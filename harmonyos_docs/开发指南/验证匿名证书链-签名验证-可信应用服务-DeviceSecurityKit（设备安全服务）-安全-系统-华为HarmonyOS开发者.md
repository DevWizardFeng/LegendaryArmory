<h1 _ngcontent-gct-c119="" class="doc-title ng-star-inserted" title="验证匿名证书链"> 验证匿名证书链 </h1>

<div _ngcontent-gct-c106="" auitextselectionexpansion="" class="markdown-body ng-star-inserted" style="position: relative;"> <div><p>为防止第三方伪造数据，端侧和云侧在对数据进行验签之前，必须先验证匿名证书链的正确性。端侧对匿名证书链的校验处理接口，请参考“<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/create-verify-cerchainvalidator-object" target="_blank">证书链校验器对象的创建和校验</a>”。云侧对匿名证书链的校验处理接口，请参考“<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/device-attestation" target="_blank">设备真实性证明</a>”的服务器端开发章节。</p> <p>初始化证明会话时返回的匿名证书链包含三级证书，验证步骤如下：</p> <ol><li><span>使用<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/huks-overview" target="_blank">Universal Keystore Kit</a>官网提供的根CA证书对匿名证书链的合法性进行校验。（<a href="https://pki.consumer.huawei.com/ca/cer/Huawei_CBG_ECC_Device_Attestation_Root_CA.cer" target="_blank">根CA证书下载地址</a>）</span><p></p><div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content"><p>请勿在应用服务器中使用子CA证书对密钥证明证书链进行校验，子CA证书可能会因为有效期结束、证书被吊销等发生变化。</p> </div></div></div> <p></p></li><li><span>解析三级证书获取应用公钥、AppID、密钥管理部件ID。</span><p></p><ol><li>应用公钥直接从密钥证明证书的subjectPublicKeyInfo字段获取。</li><li>AppID从扩展字段“1.3.6.1.4.1.2011.2.376.2.1.3.1”获取。</li><li>密钥管理部件ID从扩展字段“1.3.6.1.4.1.2011.2.376.2.2.2.6”获取。</li></ol> <p></p></li><li><span>检查三级证书的扩展字段是否符合预期：</span><p></p><ol><li>密钥管理部件ID应为“0E01669E9CFF3848A9538568F1A483E3”。</li><li>AppID的值应为调用创建证明密钥接口的应用AppID。</li></ol> <div><div class="hw-editor-tip info"><div class="title">说明</div><div class="content"><p>“应用AppID”指的是HarmonyOS应用的ID，包含bundleName和签名证书公钥的哈希值。 获取应用AppID方法请参考“<a href="https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-bundlemanager#bundlemanagergetbundleinfoforself-1" target="_blank">bundleManager 模块</a>”。</p> </div></div></div> <p></p></li></ol> </div> <div></div></div>
